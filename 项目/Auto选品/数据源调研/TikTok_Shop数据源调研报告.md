## 二、官方数据源详解

### 2.1 TikTok Shop Seller API (Shop Open Platform)

**接入方式**: TikTok开发者中心 (partner.tiktokshop.com)

| 维度 | 详情 |
|------|------|
| **认证要求** | App ID + App Secret (OAuth 2.0) |
| **当前版本** | V2（V1将于2025年12月31日完全废弃） |
| **费率限制** | 商品类: 60 req/min；订单类: 30 req/min |
| **数据范围** | 商品/库存/订单/物流/财务（全链路） |
| **合规性** | ✅ 官方授权，完全合规 |

**核心端点**:

```
POST /api/v2/product/add              # 新品注册（含SKU）
PUT  /api/v2/product/{id}              # 商品更新
POST /api/v2/product/stock/update      # 库存更新
GET  /api/v2/order/list                # 订单列表
POST /api/v2/logistics/ship            # 发货处理
GET  /api/v2/finance/transaction/list  # 交易明细
POST /api/v2/webhook/subscription      # 事件订阅
```

**API增强功能**:

| API名称 | 用途 |
|---------|------|
| Get Products SEO Words | SEO关键词建议 |
| Get Recommended Product Titles | AI标题优化 |
| Image Optimization API | 白底图转换 |
| Product Information Issue Diagnosis | listing质量诊断 |

**推荐理由**: 
- 官方唯一认可的电商ERP对接方式
- 数据实时性最强（毫秒级延迟）
- 支持完整的店铺运营闭环

### 2.2 TikTok Shop Research API

**接入方式**: 学术/研究用途申请

| 维度 | 详情 |
|------|------|
| **配额限制** | 10,000 points/hour |
| **数据范围** | 公开店铺/商品/视频数据 |
| **访问权限** | 需研究资质审核 |
| **数据类型** | 只读，不可写入 |

**核心端点**:

```
GET /v2/research/tts/shop              # 店铺统计（GMV/评分/粉丝）
GET /v2/research/tts/product           # 商品详情（价格/评论/库存）
GET /v2/research/video/query           # 视频搜索（按标签/时间段）
```

**适用场景**:
- 竞品店铺监控
- 市场大盘趋势分析
- 学术研究/报告生成

**访问地址**: TikTok Seller Center → 选品广场

### 5.2 数据同步策略

| 数据类型 | 更新频率 | 数据源 | 存储周期 |
|----------|----------|--------|----------|
| 店铺商品 | 实时 | 官方API | 永久 |
| 订单数据 | 实时 | 官方API | 永久 |
| 选品榜单 | 每日 | EchoTik/API | 24个月 |
| 达人数据 | 每周 | EchoTik | 12个月 |
| 视频数据 | 每2小时 | FastMoss | 90天 |
| 市场趋势 | 每日 | 官方+第三方 | 永久 |

### 5.3 合规注意事项

1. **数据采集合规**
   - 仅使用官方授权API和TSP资质平台
   - 禁止爬虫采集TikTok数据
   - 遵守平台Robots.txt和服务条款

2. **数据使用合规**
   - 不存储用户个人信息
   - 竞品数据仅用于分析研究
   - 遵守GDPR和各地数据法规

3. **商业使用合规**
   - EchoTik API需商业授权
   - 第三方数据不可直接转售
   - 数据分析报告需注明数据来源
