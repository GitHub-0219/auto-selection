## 概览

本文档提供面向投资人的财务预测模板，适用于中国创业公司融资场景。所有金额单位默认为人民币（万元），可根据实际情况调整。

## 收入预测模型

### SaaS订阅模式

**预测逻辑**
```
月度经常性收入(MRR) = 付费客户数 × 客单价(月)

年度经常性收入(ARR) = MRR × 12

收入预测公式：
第N月MRR = 第N-1月MRR × (1 - 流失率) + 新增MRR
```

**预测逻辑**
```
平台GMV = 交易用户数 × 单用户年交易额

平台收入 = GMV × 综合货币化率

货币化率 = (广告收入 + 佣金收入 + 增值服务收入) / GMV
```

## 关键指标计算

### 单位经济指标

**客户获取成本（CAC）**
```
CAC = 销售费用 + 市场费用 / 新增付费客户数

示例：
季度销售费用：100万元
季度市场费用：50万元
新增付费客户：50个
CAC = (100 + 50) / 50 = 3万元/客户
```

**客户终身价值（LTV）**
```
LTV = 客单价(月) × 毛利率 / 月流失率

或

LTV = 客单价(年) × 平均客户生命周期(年) × 毛利率

示例：
月客单价：5000元
毛利率：70%
月流失率：2%
LTV = 5000 × 70% / 2% = 17.5万元
```

**LTV/CAC比值**
```
LTV/CAC = 客户终身价值 / 客户获取成本

健康指标：
- LTV/CAC > 3：健康
- LTV/CAC > 5：优秀
- LTV/CAC < 3：需优化

示例：
LTV：17.5万元
CAC：3万元
LTV/CAC = 17.5 / 3 = 5.83
```

**CAC回收周期**
```
CAC回收周期 = CAC / (客单价(月) × 毛利率)

健康指标：
- < 12个月：健康
- < 18个月：可接受
- > 24个月：需关注

示例：
CAC：3万元
月客单价：5000元
毛利率：70%
回收周期 = 30000 / (5000 × 70%) = 8.6个月
```

### 增长与效率指标

**净收入留存率（NDR）**
```
NDR = (期初MRR + 增购MRR - 流失MRR - 降级MRR) / 期初MRR × 100%

健康指标：
- NDR > 100%：负流失，优秀
- NDR > 110%：卓越
- NDR < 100%：存在流失压力
```

**燃烧倍数（Burn Multiple）**
```
Burn Multiple = 年净烧钱额 / 年新增ARR

健康指标：
- < 1.0：优秀（每烧1元创造1元ARR）
- 1.0-1.5：健康
- 1.5-2.0：需关注
- > 2.0：效率较低
```

**Rule of 40**
```
Rule of 40 = 收入增长率 + 利润率

健康指标：
- > 40：健康增长
- > 50：优秀
- < 40：增长效率不足
```

### 运营指标

**获客漏斗**
```
| 阶段 | 定义 | 转化率 |
|------|------|--------|
| 曝光 | 触达用户数 | 100% |
| 访问 | 访问官网/APP | XX% |
| 注册 | 完成注册 | XX% |
| 试用 | 开始试用/演示 | XX% |
| 付费 | 完成首单 | XX% |
| 复购 | 二次购买 | XX% |

整体转化率 = 付费用户数 / 曝光用户数
```

**团队效率**
```
人均产出 = 年收入 / 团队人数
人均毛利 = 年毛利 / 团队人数
研发人效 = 研发产出 / 研发人数
销售人效 = 新签合同额 / 销售人数
```

## 情景分析框架

### 三种情景设定

**保守情景**
- 假设：市场增长放缓、竞争加剧、获客难度增加
- 应用：评估最低生存需求、融资底线

**基准情景**
- 假设：按当前趋势平稳发展、达成主要目标
- 应用：主要规划依据、投资人沟通基础

**乐观情景**
- 假设：市场加速、竞争格局优化、运营超预期
- 应用：评估上限空间、激励机制设计

## 预测注意事项

### 合理性检查

**收入预测**
- 增长率是否符合行业发展水平
- 细分市场天花板是否足够
- 获客能力是否支撑目标

**成本预测**
- 毛利率是否合理（参考行业水平）
- 人效是否可持续
- 规模效应是否体现

**现金流预测**
- 融资周期是否匹配里程碑
- 应急储备是否充足
- 季节性波动是否考虑

### 常见问题

**过度乐观**
- 避免年增长率超过200%持续多年
- 避免1年内实现盈亏平衡（早期项目）
- 避免忽略竞争加剧影响

**结构失衡**
- 避免研发投入占比过低
- 避免销售费用占比畸高
- 避免忽视管理成本增长

**逻辑不一致**
- 确保客户增长与收入增长匹配
- 确保团队扩张与业务增长匹配
- 确保现金消耗与里程碑匹配
