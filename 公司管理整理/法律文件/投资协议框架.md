1. **公司情况**：海南XX跨境电子商务有限公司（以下简称"公司"）系一家依据中国法律设立并有效存续的有限责任公司，专注于TikTok Shop东南亚市场的AI跨境电商业务
2. **投资方**：______（以下简称"投资方"）系依据______法律设立并有效存续的______（机构/自然人），有意向对公司进行投资

**第三款 反稀释保护**

| 类型 | 选择 |
|------|------|
| 完全棘轮（Full Ratchet） | □ |
| 加权平均（Weighted Average） | ■ |

加权平均计算公式：

```
调整后价格 = (调整前价格 × 已发行股份 + 实际支付总价) / 调整后股份总数
```

## 第三章 公司治理

### 第六条 董事会

**第一款 董事会组成**

| 成员 | 席位 | 指定方 |
|------|------|--------|
| 董事长 | 1 | 创始人 |
| 董事A | 1 | 创始人 |
| 董事B | 1 | 创始人 |
| 董事（投资人代表） | 1 | 投资方 |
| 独立董事 | 1 | 共同提名 |

**第二款 投资人董事权利**

投资人董事享有与其他董事同等的权利，包括：
1. 提议召开董事会
2. 在董事会上表决
3. 签署董事会决议

**第三款 观察员权**

如投资方未委派董事，投资方有权委派1名董事会观察员，列席董事会会议，但无表决权。

### 第七条 观察员权利

观察员享有以下权利：
1. 接收董事会会议通知和议程
2. 列席董事会会议
3. 获取董事会会议资料
4. 向董事会提出建议

## 第四章 创始人承诺

### 第八条 全职工作承诺

创始人应将其全部工作时间和精力投入公司，且未经投资方书面同意，不得：
1. 在公司外从事任何有报酬的工作
2. 在竞争性公司担任任何职务
3. 参与任何与公司业务相关的商业活动

### 第九条 竞业禁止

**第一款 竞业禁止期限**

创始人任职期间及离职后______个月内。

**第二款 竞业禁止范围**

| 范围 | 说明 |
|------|------|
| 地域 | 东南亚地区及中国境内 |
| 业务 | 跨境电商、TikTok Shop相关、AI选品 |

### 第十条 知识产权归属

创始人确认并同意：
1. 创始人现有及将来开发的所有知识产权归公司所有
2. 创始人应签署必要的文件配合完成知识产权转让
3. 创始人不得将公司知识产权用于非公司业务

### 第十一条 股份锁定

**第一款 锁定期**

创始人持有的股份在交割后______个月内不得转让。

**第二款 例外**

以下情形可在锁定期内转让（须经投资方书面同意）：
1. 转让给创始人全资拥有的公司
2. 因继承导致的转让
3. 因离婚财产分割导致的转让

**第二款 业绩调整**

如公司未达到业绩承诺的______%，触发估值调整机制。

### 第十三条 估值调整机制

**第一款 现金补偿**

若未达到业绩承诺，公司或创始人应向投资方进行现金补偿：

```
补偿金额 = 投资金额 × (1 - 实际业绩/承诺业绩)
```

**第二款 股份补偿**

若未达到业绩承诺，创始人应向投资方无偿转让股份：

```
补偿股份 = 投资方持股 × (1 - 实际业绩/承诺业绩)
```

**第三款 补偿上限**

现金补偿和股份补偿合计不超过投资金额的______%。

## 第六章 回购权

### 第十四条 回购触发

**第一款 强制回购触发事件**

以下任一事件发生时，投资方有权要求公司或创始人回购投资方持有的股份：

| 序号 | 事件 |
|------|------|
| 1 | 公司在______年前未完成IPO |
| 2 | 创始人严重违反本协议或公司章程 |
| 3 | 公司主营业务发生重大不利变化 |
| 4 | 公司因违法行为被吊销营业执照 |
| 5 | 公司发生清算、解散或破产 |
| 6 | 创始人离职（主动或被动）后______个月内 |

**第二款 回购价格**

| 情形 | 回购价格 |
|------|----------|
| IPO失败 | 投资金额×______倍 + 未分配股息 |
| 创始人违约 | 投资金额×______倍 |
| 其他情形 | 投资金额 + 年化______%收益 |

### 第十五条 回购程序

1. 投资方书面通知公司要求回购
2. 公司在收到通知后30日内确认回购
3. 回购款项在确认后60日内支付
4. 如公司无法回购，创始人承担连带回购责任

## 第七章 拖带权与随售权

### 第十六条 拖带权（Drag-Along）

**第一款 触发条件**

如满足以下条件，持有______%以上股份的股东可强制其他股东共同出售股份：
1. 出售价格不低于本轮投资金额的______倍
2. 该出售系善意商业交易
3. 其他股东获得同等待遇

**第二款 通知程序**

拖带权行使方应在完成出售前30日书面通知其他股东。

### 第十七条 随售权（Tag-Along）

**第一款 随售权行使**

如创始人向第三方出售股份，投资方有权按相同条件共同出售其股份。

**第二款 通知程序**

创始人在与第三方签订转让协议后5日内书面通知投资方。

## 第八章 陈述与保证

### 第十八条 公司的陈述与保证

公司向投资方陈述并保证：

1. **公司设立与存续**
   - 公司系依法设立并有效存续的有限责任公司
   - 公司有权从事营业执照载明的经营业务

2. **股权与授权**
   - 公司股权清晰，不存在代持、质押或其他权利限制
   - 公司签署本协议已获得所有必要的授权

3. **财务状况**
   - 公司财务报表真实、完整地反映了公司财务状况
   - 公司不存在未披露的重大负债或或有负债

4. **合规经营**
   - 公司经营符合法律法规
   - 公司不存在未披露的重大诉讼、仲裁或行政处罚

5. **知识产权**
   - 公司拥有的知识产权清晰、完整
   - 不存在侵犯第三方知识产权的情形

### 第十九条 创始人的陈述与保证

创始人向投资方陈述并保证：

1. 创始人具备完全民事行为能力
2. 创始人持有的公司股份为其个人合法财产
3. 创始人未从事任何与公司业务竞争的业务
4. 创始人签署本协议不会违反其与第三方的协议

### 第二十条 投资方的陈述与保证

投资方向公司和创始人陈述并保证：

1. 投资方具备签署本协议的能力
2. 投资资金来源合法
3. 投资方签署本协议不会违反其与第三方的协议

## 第九章 违约责任

### 第二十一条 违约情形

| 违约类型 | 具体情形 |
|----------|----------|
| 根本违约 | 违反本协议核心义务，导致本协议目的无法实现 |
| 一般违约 | 违反本协议其他义务 |
| 陈述保证违反 | 陈述与保证失实 |

### 第二十二条 违约救济

**第一款 违约金**

违约方应向守约方支付违约金：
- 根本违约：投资金额的______%
- 一般违约：投资金额的______%

**第二款 损害赔偿**

违约金不足以弥补损失的，违约方应继续赔偿守约方的全部损失。

**第三款 股份回购**

如创始人严重违约，投资方有权要求按本协议第十四条行使回购权。

## 第十章 争议解决

### 第二十三条 友好协商

争议发生后，各方应首先通过友好协商解决。

### 第二十四条 仲裁

协商不成时，任何一方可将争议提交：
- **仲裁机构**：中国国际经济贸易仲裁委员会（CIETAC）
- **仲裁地点**：海口
- **仲裁语言**：中文

### 第二十五条 适用法律

本协议适用中华人民共和国法律。

---

## 第十一章 其他

### 第二十六条 协议生效

本协议自各方签署且完成交割之日起生效。

### 第二十七条 交割

交割应满足以下条件：
1. 各先决条件均已满足或被豁免
2. 投资款项已足额支付至公司指定账户
3. 各方已完成股份认购相关文件的签署

### 第二十八条 协议期限

本协议有效期为公司存续期间，但估值调整条款、回购条款在本协议终止后继续有效。
