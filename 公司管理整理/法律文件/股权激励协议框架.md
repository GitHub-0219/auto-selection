## 鉴于条款

**第一条 协议背景**

1. **公司情况**：海南XX跨境电子商务有限公司（以下简称"公司"）系一家依据中国法律设立并有效存续的有限责任公司
2. **激励目的**：为建立、健全公司长效激励机制，吸引和留住优秀人才，充分调动公司核心骨干的积极性，公司决定实施股权激励计划
3. **激励依据**：本协议依据公司股东会/董事会批准的《股权激励计划》制定
4. **员工确认**：员工确认其符合公司股权激励计划的授予条件，愿意按照本协议的约定接受公司授予的股权激励

## 第一章 定义与释义

### 第二条 定义

| 术语 | 定义 |
|------|------|
| **公司** | 海南XX跨境电子商务有限公司 |
| **激励对象** | 符合股权激励计划条件并获授股权激励的员工 |
| **授予日** | 公司向激励对象授予股权激励的日期 |
| **行权期** | 激励对象可以行权的期间 |
| **行权价格** | 激励对象购买股份的价格 |
| **行权条件** | 激励对象行权需要满足的条件 |
| **成熟期** | 股份逐步解锁的期限 |
| **限制性股票** | 在满足特定条件前不得转让的股份 |
| **期权** | 购买公司股份的权利 |
| **虚拟股** | 仅享有分红权，不享有投票权的股份 |

### 第三条 股权激励类型

根据公司股权激励计划，本次授予的股权激励类型为：

| 类型 | 说明 |
|------|------|
| 期权（Stock Option） | 员工有权在满足条件后以约定价格购买公司股份 |
| 限制性股票（RSU） | 公司授予员工股份，但在满足条件前不得转让 |
| 虚拟股（Phantom Stock） | 员工仅享有分红权和增值权，不实际持有股份 |

### 第五条 股权激励的授予条件

激励对象获授股权激励，需同时满足以下条件：

1. **任职条件**
   - 截至授予日，激励对象与公司或子公司保持劳动关系
   - 激励对象未违反公司规章制度或劳动合同

2. **考核条件**
   - 激励对象最近一个考核年度绩效评定达到____级以上
   - 激励对象不存在重大违规违纪行为

3. **其他条件**
   - 激励对象已签署本协议及保密协议、竞业禁止协议
   - 激励对象不存在法律、法规或公司规定的不宜情形

## 第三章 成熟机制

| 特殊贡献 | 额外成熟____% |

| 解锁期 | 解锁时间 | 解锁比例 | 累计解锁比例 |

## 第四章 行权与行使

### 第八条 期权行权条件

**第一款 公司业绩条件**

| 解锁期 | 公司营收目标 | 公司估值目标 |
|--------|--------------|--------------|
| 第一期 | ____万元 | ____万元 |
| 第二期 | ____万元 | ____万元 |
| 第三期 | ____万元 | ____万元 |
| 第四期 | ____万元 | ____万元 |

**第二款 个人绩效条件**

激励对象最近一个考核年度个人绩效等级需达到：
- 优秀（A）
- 良好（B+）
- 合格（B）

**第三款 在职条件**

激励对象在行权时需与公司保持劳动关系。

### 第九条 行权程序

**第一款 行权申请**

1. 激励对象在行权期内，满足行权条件后，可向公司提交书面行权申请
2. 行权申请应载明拟行权的期权数量

**第二款 行权确认**

1. 公司应在收到行权申请后____个工作日内进行确认
2. 确认内容包括：行权资格、行权数量、行权价格

**第三款 行权支付**

1. 激励对象应在行权确认后____个工作日内将行权款项支付至公司指定账户
2. 行权款项 = 行权数量 × 行权价格

**第四款 股份登记**

1. 公司在收到行权款项后____个工作日内完成股份登记
2. 股份登记完成后，激励对象成为公司正式股东

### 第十条 税务处理

| 税务类型 | 税率 | 缴纳方式 |
|----------|------|----------|
| 个人所得税 | 按"工资薪金所得"或"财产转让所得"计算 | 公司代扣代缴 |
| 印花税 | 0.05% | 公司代扣代缴 |

**税务筹划建议**：
1. 符合条件地区可申请递延纳税
2. 公司可提供税务咨询服务

## 第五章 失效与回购

### 第十一条 期权失效

**第一款 失效情形**

| 失效类型 | 情形 | 处理方式 |
|----------|------|----------|
| 到期失效 | 行权期满未行权 | 全部作废 |
| 离职失效 | 激励对象主动离职 | 未成熟作废，已成熟可在____日内行权 |
| 辞退失效 | 激励对象被辞退 | 全部作废 |
| 死亡失效 | 激励对象死亡 | 已成熟可继承，未成熟作废 |

**第二款 离职后处理**

| 离职原因 | 已成熟期权 | 未成熟期权 |
|----------|------------|------------|
| 主动离职 | ____日内可按原价行权 | 作废 |
| 被动离职（公司原因） | ____日内可按原价行权 | 作废 |
| 严重违规被辞退 | 须返还收益 | 作废 |
| 死亡/丧失行为能力 | 可继承，按原计划行权 | 加速成熟____% |

### 第十二条 回购机制

**第一款 回购触发**

公司有权在以下情形回购激励对象持有的股份：

| 情形 | 回购价格 |
|------|----------|
| 激励对象违反竞业禁止义务 | 激励对象已支付行权价格的____% |
| 激励对象违反保密义务 | 激励对象已支付行权价格的____% |
| 激励对象严重损害公司利益 | 无偿回购 |
| 激励对象在IPO前离职（特定情形） | 公允价值×____% |

**第二款 回购程序**

1. 公司向激励对象发出回购通知
2. 激励对象在收到通知后____日内配合完成股份转让
3. 公司在完成转让后____日内支付回购款项

## 第六章 竞业限制与保密

### 第十三条 竞业限制

**第一款 竞业禁止期限**

激励对象在任职期间及离职后____个月内，不得从事与公司业务相竞争的工作。

**第二款 竞业禁止范围**

| 范围 | 说明 |
|------|------|
| 地域 | 中国大陆及东南亚地区 |
| 业务 | 跨境电商、TikTok Shop相关、AI选品系统 |

**第三款 违约后果**

激励对象违反竞业限制的：
1. 公司有权回购全部股份
2. 激励对象应向公司支付违约金____万元
3. 激励对象已行权收益归公司所有

### 第十四条 保密义务

激励对象应遵守公司的保密制度，不得泄露公司商业秘密。

## 第七章 争议解决

### 第十五条 协商

争议发生后，各方应首先通过友好协商解决。

### 第十六条 仲裁

协商不成的，任何一方可将争议提交公司所在地仲裁委员会仲裁。

---

## 第八章 其他

### 第十七条 协议变更

本协议的修改须经公司书面同意，并签署书面变更协议。

### 第十八条 协议终止

**第一款 自然终止**

本协议在以下情形自然终止：
1. 期权全部行权或失效
2. 限制性股票全部解锁或回购

**第二款 提前终止**

本协议可在以下情形提前终止：
1. 激励对象与公司协商一致
2. 公司股权激励计划终止
3. 法律、法规规定的其他情形

### 第十九条 通知

各方在本协议中列明的联系地址为有效送达地址。

### 第二十条 协议份数

本协议一式贰份，公司留存一份，激励对象留存一份，具有同等法律效力。
