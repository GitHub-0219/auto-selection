# 用户画像

## 一、用户概述

### 1.1 目标用户定义
我们的目标用户是跨境电商卖家，主要集中在年销售额50万至5000万人民币的中小型卖家群体。

### 1.2 用户规模估算
| 细分市场 | 年销售额 | 用户规模（估算） | 占比 |
|---------|---------|----------------|------|
| 初创型 | 50-200万 | 10万+ | 60% |
| 成长型 | 200-1000万 | 5万+ | 30% |
| 规模型 | 1000-5000万 | 1万+ | 10% |

## 四、用户决策分析

### 4.1 购买决策流程
```
问题识别 → 信息搜索 → 方案评估 → 购买决策 → 购后评估
```

| 阶段 | 用户行为 | 我们能做的 |
|-----|---------|---------|
| 问题识别 | 认识到选品困难 | 内容营销触达 |
| 信息搜索 | 搜索比较选品工具 | SEO、内容营销 |
| 方案评估 | 对比功能价格 | 免费试用、案例 |
| 购买决策 | 决定是否付费 | 价值展示、促销 |
| 购后评估 | 使用后评价 | 客服跟进、服务 |

### 4.2 购买考虑因素
| 因素 | 重要性 | 说明 |
|-----|-------|-----|
| 功能是否满足 | ⭐⭐⭐⭐⭐ | 最核心的考量 |
| 数据准确性 | ⭐⭐⭐⭐⭐ | 核心价值 |
| 价格 | ⭐⭐⭐⭐ | 重要但非首要 |
| 易用性 | ⭐⭐⭐⭐ | 影响使用意愿 |
| 客服支持 | ⭐⭐⭐ | 影响长期使用 |
| 品牌口碑 | ⭐⭐⭐ | 影响初次信任 |
| 同行推荐 | ⭐⭐ | 口碑传播 |

### 4.3 价格敏感度
| 用户类型 | 价格敏感度 | 愿意支付 | 关注点 |
|---------|-----------|---------|-------|
| 初创型 | 高 | 100-200元/月 | 性价比 |
| 成长型 | 中 | 200-500元/月 | 价值 |
| 规模型 | 低 | 500-2000元/月 | 效率 |

## 六、用户生命周期

### 6.1 用户生命周期阶段
```
潜在用户 → 新用户 → 活跃用户 → 付费用户 → 忠诚用户
```

| 阶段 | 定义 | 关键指标 | 运营重点 |
|-----|-----|---------|---------|
| 潜在用户 | 了解但未注册 | 注册率 | 激活转化 |
| 新用户 | 注册但未使用 | 激活率 | 引导上手 |
| 活跃用户 | 经常使用 | 活跃度 | 价值传递 |
| 付费用户 | 转化为付费 | 转化率 | 续费留存 |
| 忠诚用户 | 长期付费 | 续费率 | 推荐传播 |

### 6.2 各阶段运营策略
| 阶段 | 策略 | 行动 |
|-----|-----|-----|
| 潜在用户 | 认知建立 | 内容营销、广告 |
| 新用户 | 快速上手 | 新手引导、教程 |
| 活跃用户 | 深度使用 | 功能引导、案例 |
| 付费用户 | 持续价值 | 定期回访、升级 |
| 忠诚用户 | 推荐传播 | 推荐激励、社群 |

### 7.2 激活策略
| 策略 | 说明 |
|-----|-----|

|-----|-----|

| 社群运营 | 建立用户社群，增强粘性 |
