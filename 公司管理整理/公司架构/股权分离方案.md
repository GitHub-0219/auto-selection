# 科技管理集团 - 股权分离方案

## 一、股权分离设计原则

### 1.1 核心原则
1. **所有权与控制权分离**：投票权集中，分红权可分散
2. **短期激励与长期激励结合**：期权池+动态分配
3. **风险与收益对等**：不同角色承担不同风险，享受不同收益
4. **预留调整空间**：为未来融资、团队扩张预留股权

### 1.2 分离维度
| 维度 | 分离方式 | 适用场景 |
|------|----------|----------|
| 投票权 vs 分红权 | AB股结构 | 创始人保持控制权 |
| 实际股权 vs 经济权利 | 股权激励池 | 员工激励 |
| 公司股权 vs 经营分红 | 阶梯式分红 | 核心团队绑定 |

## 二、初始股权分配方案

### 2.1 创始人股权分配（示例）

假设创始团队3人：CEO（技术）、CTO、COO

| 角色 | 股权比例 | 投票权（AB股） | 分红权 | 备注 |
|------|----------|---------------|--------|------|
| CEO | 60% | A类股份10倍投票权 | 60% | 核心创始人 |
| CTO | 20% | A类股份10倍投票权 | 20% | 联合创始人 |
| COO | 10% | A类股份10倍投票权 | 10% | 联合创始人 |
| 期权池 | 10% | - | 10% | 预留 |

**合计**：100%

### 2.2 股权激励池设计

#### 2.2.1 池子规模
- **天使轮融资前**：10-15%
- **A轮融资前**：15-20%
- **B轮融资前**：20-25%

#### 2.2.2 分配对象
| 对象 | 建议比例 | 分配方式 |
|------|----------|----------|
| 核心管理层 | 30-40% | 期权/限制性股票 |
| 技术骨干 | 30-40% | 期权/限制性股票 |
| 早期员工 | 20-30% | 期权 |
| 顾问/顾问 | 5-10% | 期权 |

#### 2.2.3 行权条件
- **时间维度**：4年成熟，1年等待期，按月或按季度成熟
- **业绩维度**：公司估值达到目标、营收达标
- **留任条件**：在职方可行权，离职则未成熟部分作废

## 三、AB股结构设计

### 3.1 AB股模式

| 股份类型 | 持有者 | 投票权 | 分红权 |
|----------|--------|--------|--------|
| A类股份 | 创始人 | 10票/股 | 与持股比例一致 |
| B类股份 | 投资人/员工 | 1票/股 | 与持股比例一致 |

### 3.2 投票权分配表（示例）

| 股东 | A类股数 | B类股数 | 实际投票权 | 经济权益 |
|------|---------|---------|-----------|----------|
| 创始人A | 600 | 0 | 6000票 | 60% |
| 创始人B | 200 | 0 | 2000票 | 20% |
| 创始人C | 100 | 0 | 1000票 | 10% |
| 投资人 | 0 | 100 | 100票 | 10% |

**实际控制权**：创始人团队 90%（按票数），经济权益 90%

### 3.3 AB股转换机制
1. **自动转换**：创始人转让股份给非创始人时，A类股自动转为B类股
2. **死亡/失去行为能力**：A类股自动转为B类股
3. **上市要求**：上市时可考虑同股同权（香港/美国可保留AB股）

### 3.4 法律依据
- **中国境内**：有限责任公司可约定章程条款实现类似AB股效果
- **开曼/BVI**：公司法明确支持AB股结构
- **VIE架构**：通过WFOE实现实际控制

## 四、投票权与分红权分离设计

### 4.1 分离架构

```
┌─────────────────────────────────────┐
│           控制层（投票权）            │
│  创始人通过持股公司HoldCo控股         │
│  享有AB股优势                        │
└─────────────────────────────────────┘
                │
                ▼
┌─────────────────────────────────────┐
│         经济层（分红权）             │
│  投资人、员工通过持股平台参与        │
│  按比例享有分红                      │
└─────────────────────────────────────┘
```

### 4.2 员工持股平台设计

#### 4.2.1 有限合伙企业（LP）模式
```
┌─────────────────────────────────────┐
│     XX员工持股合伙企业（有限合伙）   │
│                                     │
│  GP（普通合伙人）：创始人            │
│  - 执行合伙事务                      │
│  - 行使投票权                        │
│  - 承担无限责任（风险可控）          │
│                                     │
│  LP（有限合伙人）：员工              │
│  - 享有分红权                        │
│  - 不参与管理                        │
│  - 以出资额为限承担责任              │
└─────────────────────────────────────┘
         │ 持有股权
         ▼
┌─────────────────────────────────────┐
│         持股公司（HoldCo）            │
└─────────────────────────────────────┘
```

#### 4.2.2 优势
- 创始人通过GP控制持股平台投票权
- 员工享受分红权但不参与经营决策
- 避免因员工离职导致股权分散
- 税务相对优惠（避免双重征税）

### 4.3 分红权分配方案（示例）

| 分配对象 | 分红比例 | 考核指标 |
|----------|----------|----------|
| 创始人A | 50% | 公司整体业绩 |
| 创始人B | 15% | 技术部门业绩 |
| 创始人C | 10% | 运营部门业绩 |
| 员工持股池 | 20% | 个人KPI+公司业绩 |
| 期权池 | 5% | - |

## 五、股权动态调整机制

### 5.1 调整触发条件

| 触发情形 | 调整方向 | 调整幅度 |
|----------|----------|----------|
| 核心离职 | 回购/冻结 | 100% |
| 业绩不达标 | 调减 | 20-50% |
| 贡献超预期 | 增发 | 10-30% |
| 融资稀释 | 整体稀释 | 按融资条款 |

### 5.2 回购机制

#### 5.2.1 回购情形
- 员工离职（主动或被动）
- 创始人失去行为能力
- 公司合并/被收购

#### 5.2.2 回购价格
| 情形 | 回购价格 |
|------|----------|
| 正常离职 | 公允估值×已成熟比例 |
| 被辞退 | 成本价 |
| 创始人死亡 | 公允估值 |
| 合并收购 | 并购价格 |

### 5.3 重新分配机制

1. **年度回顾**：每年评估股权合理性
2. **贡献度评估**：引入360度评估机制
3. **公平性保障**：设置异议处理流程

## 六、融资股权稀释保护

### 6.1 融资轮次与稀释

| 融资轮次 | 估值区间 | 股权出让 | 稀释幅度 |
|----------|----------|----------|----------|
| 天使轮 | 1000-3000万 | 10-20% | 创始人稀释10-20% |
| A轮 | 5000万-1亿 | 15-25% | 累计稀释25-40% |
| B轮 | 2-5亿 | 15-20% | 累计稀释40-55% |
| C轮及以上 | 10亿+ | 10-15% | 累计稀释50-65% |

### 6.2 反稀释条款

#### 6.2.1 完全棘轮（Full Ratchet）
```
若后续融资估值低于当前投资价，投资人按最新估值调整持股
（不利于创始人，慎用）
```

#### 6.2.2 加权平均（Weighted Average）
```
考虑后续融资金额，按加权平均调整
（相对公平，推荐）
```

#### 6.2.3 保护方式
- 创始人：设定最低股权比例红线（如创始人不少于40%）
- 投资人：享有优先认购权（可按比例跟投）

### 6.3 优先清算权

| 清算优先级 | 倍数 | 适用对象 |
|-----------|------|----------|
| 1倍优先 | 1x | 早期投资人 |
| 1.5倍优先 | 1.5x | 后期投资人 |
| 参与分配 | 参与剩余分配 | 全部投资人 |

## 七、退出机制设计

### 7.1 退出路径

| 路径 | 时间节点 | 创始人收益 |
|------|----------|-----------|
| 并购退出 | B轮/C轮后 | 高（一次性） |
| 上市退出 | C轮后 | 最高（但受锁定期限制） |
| 管理层回购 | 长期运营 | 中等（现金压力） |
| 股权转让 | 随时 | 取决于估值 |

### 7.2 锁定期安排

| 股东类型 | 锁定期 | 解锁方式 |
|----------|--------|----------|
| 创始人 | 1-3年 | 分期解锁 |
| 早期投资人 | 180天-1年 | 一次性解锁 |
| 员工 | 4年成熟 | 按月解锁 |
| 后期投资人 | 180天 | 一次性解锁 |

### 7.3 优先购买权

- 创始人享有优先购买权
- 投资人享有按比例优先购买权
- 转让需提前通知并获得同意

## 八、税务筹划

### 8.1 个人所得税

| 收入类型 | 税率 | 优惠政策 |
|----------|------|----------|
| 工资薪金 | 3-45% | 海南自贸港最高15% |
| 股息红利 | 20% | - |
| 期权行权 | 20%（财产转让所得） | 递延纳税（试点地区） |

### 8.2 企业所得税

| 企业类型 | 税率 | 优惠政策 |
|----------|------|----------|
| 高新企业 | 15% | - |
| 海南自贸港企业 | 15% | - |
| 小微企业 | 5%/10% | - |

### 8.3 税务优化建议

1. **期权递延纳税**：利用试点地区政策
2. **分红合理分配**：考虑工资+分红组合
3. **海南自贸港**：享受个人所得税优惠
4. **持股平台**：避免双重征税

## 九、实施步骤

### 9.1 第一阶段（筹备期）

1. 确定创始团队和初始股权分配
2. 设立持股公司HoldCo
3. 设立有限合伙企业作为持股平台
4. 完成工商注册

### 9.2 第二阶段（早期运营）

1. 与核心成员签订股权激励协议
2. 设置期权池（10-15%）
3. 制定股权管理制度

### 9.3 第三阶段（引入投资）

1. 与投资人协商股权分配
2. 设置反稀释条款
3. 签署股东协议

### 9.4 第四阶段（扩张期）

1. 定期评估股权合理性
2. 动态调整分配方案
3. 为上市准备

## 十、风险提示与注意事项

### 10.1 法律风险

| 风险 | 规避措施 |
|------|----------|
| AB股合法性争议 | 在章程中明确约定 |
| 股权回购纠纷 | 提前约定回购条款 |
| 税务合规风险 | 聘请专业顾问 |

### 10.2 实施风险

| 风险 | 规避措施 |
|------|----------|
| 核心成员离职影响 | 设置成熟期和回购机制 |
| 股权过于集中 | 引入职业经理人 |
| 融资稀释过多 | 设定最低持股红线 |

### 10.3 其他建议

1. **书面记录**：所有股权变更必须书面记录
2. **定期复盘**：每年审查股权分配合理性
3. **法律审查**：重大股权变更需律师审查
4. **税务申报**：按时申报个人所得税
