### 1.2 股东（大）会

#### 1.2.1 组成与地位

| 项目 | 说明 |
|------|------|
| **机构性质** | 公司的最高权力机构 |
| **组成人员** | 全体股东 |
| **法定人数** | 股份有限公司应有1人以上200人以下发起人，需有过半数在中国境内有住所 |

#### 1.2.2 主要职权

1. **投资经营决策**
   - 决定公司的经营方针和投资计划
   - 审议批准公司的年度财务预算方案、决算方案

2. **人事权**
   - 选举和更换非由职工代表担任的董事、监事
   - 决定有关董事、监事的报酬事项

3. **财务权**
   - 审议批准公司的利润分配方案和弥补亏损方案
   - 审议批准董事会报告、监事会报告

4. **重大事项决策**
   - 对公司增加或减少注册资本作出决议
   - 对发行公司债券作出决议
   - 对公司合并、分立、解散、清算或变更公司形式作出决议
   - 修改公司章程

#### 1.2.3 会议制度

| 会议类型 | 召开频率 | 通知期限 |
|----------|----------|----------|
| **年度股东大会** | 每年召开一次 | 会议召开20日前通知 |
| **临时股东大会** | 应当自事实发生之日起2个月内召开 | 会议召开15日前通知 |

**临时股东大会召开条件**：
- 董事人数不足法定人数或章程规定人数的2/3时
- 公司未弥补亏损达实收股本总额的1/3时
- 单独或合计持有公司10%以上股份的股东请求时
- 董事会认为必要时
- 监事会提议召开时
- 章程规定的其他情形

### 1.3 董事会

#### 1.3.1 组成与设置

| 项目 | 说明 |
|------|------|
| **成员人数** | 5-19人（可设董事长1人，副董事长1-2人） |
| **产生方式** | 由股东大会选举产生 |
| **董事任期** | 每届不超过3年，可连选连任 |
| **职工代表** | 可以有公司职工代表（非必须） |

#### 1.3.2 主要职权

1. **召集与报告**
   - 召集股东大会，并向股东大会报告工作
   - 执行股东大会的决议

2. **经营决策**
   - 决定公司的经营计划和投资方案
   - 制订公司年度财务预算方案、决算方案
   - 制订公司利润分配方案和弥补亏损方案
   - 制订公司增加或减少注册资本以及发行债券的方案
   - 制订公司合并、分立、解散或变更公司形式的方案

3. **人事权**
   - 决定聘任或解聘公司经理及其报酬事项
   - 根据经理的提名，决定聘任或解聘副经理、财务负责人及其报酬事项

4. **管理权**
   - 决定公司内部管理机构的设置
   - 制定公司的基本管理制度
   - 制定公司的具体规章

#### 1.3.3 董事会专门委员会（可选设置）

建议设立以下专门委员会：

| 委员会名称 | 人数要求 | 主要职责 |
|------------|----------|----------|
| **审计委员会** | 3人以上 | 财务审计、内部控制、风险管理 |
| **薪酬与考核委员会** | 3人以上 | 董事高管薪酬、绩效考核 |
| **提名委员会** | 3人以上 | 董事候选人提名、人才培养 |

---

### 1.4 监事会

#### 1.4.1 组成与设置

| 项目 | 说明 |
|------|------|
| **成员人数** | 3人以上（可设监事会主席1人，副主席1人） |
| **人员构成** | 必须包括股东代表和职工代表（职工代表比例不低于1/3） |
| **职工代表产生** | 由公司职工通过职工代表大会、职工大会或其他形式民主选举产生 |
| **监事会任期** | 每届3年，可连选连任 |
| **禁止兼任** | 董事、高级管理人员不得兼任监事 |

#### 1.4.2 主要职权

1. **财务监督**
   - 检查公司财务

2. **合规监督**
   - 对董事、高级管理人员执行公司职务的行为进行监督
   - 对违反法律、行政法规、公司章程或股东大会决议的董事、高级管理人员提出罢免的建议

3. **纠正权**
   - 当董事、高级管理人员的行为损害公司利益时，要求其予以纠正

4. **会议召集权**
   - 提议召开临时股东大会
   - 向股东大会提出提案

5. **诉讼权**
   - 可以对公司董事、高级管理人员提起诉讼

#### 1.4.3 会议制度

| 项目 | 说明 |
|------|------|
| **定期会议** | 每6个月至少召开一次 |
| **临时会议** | 监事可以提议召开 |
| **表决要求** | 决议应当经全体监事的过半数通过 |
| **表决权** | 一人一票 |

---

### 1.5 管理层（总经理及高级管理人员）

#### 1.5.1 总经理

| 项目 | 说明 |
|------|------|
| **产生方式** | 由董事会决定聘任或解聘 |
| **任期** | 由董事会决定 |
| **机构设置** | 可以设总经理1人，副总经理若干人 |

**主要职权**：
- 主持公司的生产经营管理工作
- 组织实施董事会决议
- 组织实施公司年度经营计划和投资方案
- 拟订公司内部管理机构设置方案
- 拟订公司的基本管理制度
- 制定公司的具体规章
- 提请聘任或解聘副经理、财务负责人
- 聘任或解聘除应由董事会聘任或解聘以外的负责管理人员
- 公司章程或董事会授予的其他职权

#### 1.5.2 财务负责人

| 项目 | 说明 |
|------|------|
| **产生方式** | 由董事会根据经理提名决定聘任或解聘 |
| **主要职责** | 负责公司财务管理、会计核算、预算管理等工作 |

#### 1.5.3 董事会秘书（股份有限公司必须设置）

| 项目 | 说明 |
|------|------|
| **设置要求** | 股份有限公司必须设董事会秘书 |
| **主要职责** | 负责公司股东大会和董事会会议的筹备、文件保管、信息披露等 |

### 1.6 部门设置建议

根据AI跨境电商业务特点，建议设置以下部门：

| 部门 | 负责人 | 主要职责 |
|------|--------|----------|
| **技术部** | CTO | AI技术研发、系统架构、数据分析 |

| **人事行政部** | HR总监 | 人力资源、行政后勤、法务合规 |

## 2. 股权结构设计

### 2.1 注册资本建议

| 考量因素 | 建议 |
|----------|------|
| **业务需求** | AI跨境电商业务初期运营成本估算 |
| **融资规划** | 预留后续融资稀释空间 |
| **资质要求** | 部分电商平台入驻资质要求 |
| **建议金额** | **人民币500万元-1000万元** |

**说明**：
- 2024年《公司法》已取消股份有限公司注册资本最低限额的强制要求
- 建议根据实际业务需求和未来融资规划确定
- 注册资金过高可能导致实缴压力，过低可能影响业务开展

### 2.2 股份分配方案

#### 2.2.1 初创期股权结构（建议）

| 股东类别 | 持股比例 | 股份性质 | 说明 |
|----------|----------|----------|------|
| **创始人** | 70% | 创始人股 | 创始团队控股，保障控制权 |
| **联合创始人/核心团队** | 10% | 创始团队股 | 核心高管激励 |
| **股权激励池（ESOP）** | 15% | 期权/限制性股权 | 预留未来人才激励 |
| **天使/早期投资人** | 5% | 优先股（可选） | 引入资源型投资人 |

#### 2.2.2 股权稀释规划

```
                    初创期          A轮融资后      B轮融资后
                    ─────          ────────      ───────
创始人持股         70%    →       56%     →      44.8%
联合创始人/核心     10%    →        8%     →       6.4%
ESOP               15%    →       15%     →       15%
天使轮              5%    →        4%     →       3.2%
A轮投资人           --          17%     →      13.6%
B轮投资人           --          --          16.8%
────────────────────────────
累计稀释           100%          100%          100%
```

#### 2.2.3 股份类型设计

| 股份类型 | 权利配置 | 适用对象 |
|----------|----------|----------|
| **普通股** | 投票权1:1，分红权1:1 | 创始人、核心团队 |
| **优先股** | 优先清算权、优先分红权、反稀释保护 | 外部投资人 |
| **限制性股权** | 附带服务期/业绩条件的完整股权 | 核心高管 |

### 2.3 股权激励池（ESOP）

#### 2.3.1 激励池设置

| 项目 | 建议 | 说明 |
|------|------|------|
| **激励池比例** | 总股本的10%-20% | 建议初创期15%，根据融资节奏调整 |
| **激励工具** | 期权为主，限制性股权为辅 | 初创期以期权为主，降低员工风险 |
| **持股方式** | 有限合伙持股平台 | 创始人担任GP，保障控制权 |

#### 2.3.2 激励对象分层

| 层级 | 人员范围 | 建议授予比例 | 激励工具 |
|------|----------|--------------|----------|
| **核心决策层** | 联合创始人、高管 | 60%-70% | 期权+限制性股权 |
| **中层骨干层** | 部门负责人、技术骨干 | 20%-30% | 期权 |
| **未来潜力层** | 高潜力员工 | 10% | 期权 |

#### 2.3.3 成熟机制设计

| 机制类型 | 设计要点 | 说明 |
|----------|----------|------|
| **成熟期** | 4年 | 标准成熟周期 |
| **悬崖期** | 12个月 | 满1年后一次性成熟25% |
| **成熟节奏** | 匀速成熟 | 每月成熟1/48 |
| **加速成熟** | 特殊情况 | 允许事件触发加速 |

#### 2.3.4 退出条款设计

| 退出情形 | 处理方式 |
|----------|----------|
| **主动离职** | 未成熟期权失效；已成熟期权按行权价或约定价格回购 |
| **被动离职（裁员）** | 已成熟期权加速成熟50%，按市价回购 |
| **过错离职** | 未成熟期权失效；已成熟期权按成本价回购 |
| **死亡/残疾** | 已成熟期权立即归属，继承人可选择行权或回购 |
| **公司被收购** | 期权加速成熟或替换为收购方等价权益 |
| **公司上市** | 期权可按市场规则行权 |

### 2.4 股份发行方案

#### 2.4.1 股份发行方式

| 发行方式 | 说明 | 适用场景 |
|----------|------|----------|
| **发起设立** | 发起人认购全部股份 | 初创期，股东人数较少 |
| **募集设立** | 发起人认购部分，其余向社会募集 | 需要引入多方投资者 |

#### 2.4.2 股份面值与价格

| 项目 | 建议 |
|------|------|
| **每股面值** | 人民币1元 |
| **发行价格** | 根据公司估值确定 |
| **注册资本** | 500万元（可调整） |
| **股份总数** | 500万股（可调整） |

#### 2.4.3 股份登记与备案

1. **工商登记**：股份有限公司应在公司登记机关登记注册资本、已发行的股份数
2. **股东名册**：公司应置备股东名册，记载股东信息
3. **股份托管**：考虑将股份托管至具有资质的股权托管机构

## 3. 公司治理结构

### 3.1 董事会议事规则

#### 3.1.1 会议类型与召开频率

| 会议类型 | 召开频率 | 通知期限 |
|----------|----------|----------|
| **定期会议** | 每年至少2次（上下半年各1次） | 会议召开10日前通知全体董事和监事 |
| **临时会议** | 根据需要召开 | 另定通知方式和时限 |

#### 3.1.2 临时会议提议权

- 代表1/10以上表决权的股东
- 1/3以上董事
- 监事会

#### 3.1.3 会议召集与主持

```
召集顺序：
1. 董事长召集并主持
2. 董事长不能履行职务或不履行职务 → 副董事长召集和主持
3. 副董事长不能履行职务或不履行职务 → 过半数董事共同推举一名董事召集和主持
```

#### 3.1.4 会议出席要求

| 项目 | 要求 |
|------|------|
| **有效开会条件** | 过半数董事出席 |
| **普通决议通过条件** | 全体董事过半数通过 |
| **特别决议通过条件** | 2/3以上董事出席（重大事项） |
| **表决权** | 一人一票 |

#### 3.1.5 重大事项范围（建议采用特别决议）

1. 公司章程修改
2. 公司合并、分立、解散或变更公司形式
3. 重大资产重组
4. 股权激励计划
5. 董事会成员变动
6. 超过净资产10%的投资
7. 对外担保（单项超过净资产5%）
8. 关联交易（单项超过净资产1%或年度累计超过净资产3%）

#### 3.1.6 委托出席规则

- 董事因故不能出席，可书面委托其他董事代为出席
- 委托书应当载明授权范围
- 独立董事不得委托非独立董事代为出席
- 一名董事不得接受超过两名董事的委托

#### 3.1.7 会议记录与决议

- 董事会应当对所议事项的决定作成会议记录
- 出席会议的董事应当在会议记录上签名
- 董事应当对董事会决议承担责任
- 董事对决议表决时表明异议并记载于会议记录的，可以免除责任

### 3.2 股东大会议事规则

#### 3.2.1 会议类型

| 会议类型 | 召开频率 | 通知期限 |
|----------|----------|----------|
| **年度股东大会** | 每年召开一次 | 会议召开20日前通知 |
| **临时股东大会** | 必要时召开 | 会议召开15日前通知 |

#### 3.2.2 临时股东大会召开条件

1. 董事人数不足法定人数（5人）或章程规定人数的2/3时
2. 公司未弥补亏损达实收股本总额的1/3时
3. 单独或合计持有公司10%以上股份的股东请求时
4. 董事会认为必要时
5. 监事会提议召开时
6. 章程规定的其他情形

#### 3.2.3 会议召集与主持

```
召集顺序：
1. 董事会召集，董事长主持
2. 董事长不能履行职务或不履行职务 → 副董事长主持
3. 副董事长不能或不履行职务 → 过半数董事共同推举一名董事主持
4. 董事会不履行召集义务 → 监事会召集和主持
5. 监事会不召集 → 连续90日以上单独或合计持有10%以上股份的股东自行召集
```

#### 3.2.4 提案规则

| 提案主体 | 提案要求 |
|----------|----------|
| **董事会** | 提交年度股东大会的临时提案，应在发出股东大会通知前公告 |
| **监事会** | 提议召开临时股东大会 |
| **股东** | 单独或合计持有3%以上股份的股东，可在股东大会召开10日前提出临时提案 |
| **独立董事** | 征集投票权 |

#### 3.2.5 表决方式与通过条件

| 决议类型 | 通过条件 |
|----------|----------|
| **普通决议** | 出席股东大会股东所持表决权过半数通过 |
| **特别决议** | 出席股东大会股东所持表决权2/3以上通过 |

**特别决议事项**：
1. 修改公司章程
2. 增加或减少注册资本
3. 公司合并、分立、解散或变更公司形式
4. 发行优先股
5. 章程规定的其他重大事项

#### 3.2.6 累积投票制

选举董事或监事时，可以实行累积投票制：
- 每一位出席股东大会的股东所持有的投票权 = 其持有的股份数 × 待选董事（监事）人数
- 股东可以集中投票给一名候选人，也可以分散投票

#### 3.2.7 会议记录

股东大会应当对所议事项的决定作成会议记录，记载：
- 会议时间、地点
- 出席股东人数及所持股份比例
- 审议事项及表决结果
- 出席股东签名

---

### 3.3 监事会议事规则

#### 3.3.1 会议类型与召开频率

| 会议类型 | 召开频率 | 通知要求 |
|----------|----------|----------|
| **定期会议** | 每6个月至少召开一次 | 提前通知全体监事 |
| **临时会议** | 监事可以提议召开 | 及时通知 |

#### 3.3.2 会议召集与主持

```
召集顺序：
1. 监事会主席召集和主持
2. 监事会主席不能履行职务或不履行职务 → 监事会副主席召集和主持
3. 监事会副主席不能履行职务或不履行职务 → 过半数监事共同推举一名监事召集和主持
```

#### 3.3.3 表决要求

| 项目 | 要求 |
|------|------|
| **有效开会条件** | 无特别规定，建议参照董事会 |
| **决议通过条件** | 全体监事的过半数通过 |
| **表决权** | 一人一票 |

#### 3.3.4 会议记录

- 监事会应当对所议事项的决定作成会议记录
- 出席会议的监事应当在会议记录上签名
- 监事列席董事会会议时，应当对记录情况进行核实

---

### 3.4 总经理工作细则

#### 3.4.1 职责定位

- 总经理是公司日常经营管理的总负责人
- 对董事会负责，执行董事会的各项决议
- 列席董事会会议

#### 3.4.2 主要职责

**经营管理职责**：
1. 主持公司的生产经营管理工作
2. 组织实施董事会决议
3. 组织实施公司年度经营计划和投资方案
4. 拟订公司内部管理机构设置方案
5. 拟订公司的基本管理制度
6. 制定公司的具体规章

**人事管理职责**：
1. 提请聘任或解聘公司副经理、财务负责人
2. 聘任或解聘除应由董事会聘任或解聘以外的负责管理人员
3. 负责对公司员工的考核、奖惩

**报告职责**：
1. 定期向董事会报告工作
2. 根据董事会要求提交相关报告
3. 向股东大会报告相关事项

#### 3.4.3 授权范围

| 事项类型 | 授权额度 | 说明 |
|----------|----------|------|
| **日常经营支出** | 单笔50万元以内 | 需在年度预算范围内 |
| **一般合同签署** | 单笔100万元以内 | 需在董事会授权范围内 |
| **人事任免** | 部门经理级以下 | 副经理、财务负责人除外 |
| **对外投资** | 单笔500万元以内 | 需在董事会授权范围内 |

#### 3.4.4 报告机制

| 报告类型 | 报告对象 | 报告频率 |
|----------|----------|----------|
| **经营情况报告** | 董事会 | 每月度 |
| **财务分析报告** | 董事会 | 每季度 |
| **年度工作总结与计划** | 董事会、股东大会 | 每年 |
| **重大事项报告** | 董事会 | 及时报告 |

## 4. 财务管理制度

### 4.1 财务审批流程

#### 4.1.1 费用审批层级

| 费用类型 | 金额范围 | 审批人 |
|----------|----------|--------|
| **日常费用** | 1万元以下 | 部门负责人 |
| **一般支出** | 1万-10万元 | 总经理 |
| **重大支出** | 10万-50万元 | 总经理 + CFO联签 |
| **特别支出** | 50万元以上 | 董事会决议 |

#### 4.1.2 付款审批流程

```
费用发生 → 部门负责人初审 → 财务复核 → 逐级审批 → 出纳付款 → 账务处理
                              ↓
                        预算核对/发票审核
```

#### 4.1.3 借款与预支

| 类型 | 额度限制 | 审批要求 | 还款期限 |
|------|----------|----------|----------|
| **个人备用金** | 2万元以内 | 总经理审批 | 30日内冲账 |
| **业务预支** | 10万元以内 | 总经理审批 | 业务完成后15日内 |
| **大额借款** | 10万元以上 | 董事会审批 | 按约定 |

#### 4.1.4 合同付款管理

| 合同类型 | 付款节点 | 审批要求 |
|----------|----------|----------|
| **采购合同** | 按合同约定 | 财务核对后付款 |
| **投资合同** | 按合同约定 | 董事会决议 |
| **费用类合同** | 服务完成后 | 按审批层级 |

---

### 4.2 资金管理制度

#### 4.2.1 现金管理

| 项目 | 要求 |
|------|------|
| **库存现金限额** | 原则上不超过5万元 |
| **现金使用范围** | 员工工资、零星开支等 |
| **现金保管** | 出纳人员负责，确保安全 |
| **盘点制度** | 每月至少盘点一次 |

#### 4.2.2 银行存款管理

| 项目 | 要求 |
|------|------|
| **账户开立** | 需经董事会批准 |
| **印鉴管理** | 财务章、法人章分开保管 |
| **对账制度** | 每月与银行对账，编制余额调节表 |
| **U盾管理** | 分级管理，限额控制 |

#### 4.2.3 资金调拨

| 类型 | 金额 | 审批要求 |
|------|------|----------|
| **日常调拨** | 100万元以内 | CFO审批 |
| **大额调拨** | 100万-500万元 | 总经理 + CFO联签 |
| **超大额调拨** | 500万元以上 | 董事会决议 |

#### 4.2.4 外汇管理（跨境电商适用）

| 项目 | 要求 |
|------|------|
| **账户管理** | 开立外币结算账户 |
| **结售汇** | 按需办理，关注汇率波动 |
| **跨境支付** | 通过合规渠道，确保资金安全 |
| **合规申报** | 按时完成外汇申报 |

#### 4.2.5 资金安全控制

1. **不相容职务分离**：出纳与会计分离、审批与执行分离
2. **定期对账**：每日核对、月末盘点、季度审计
3. **预警机制**：建立资金余额预警系统
4. **应急储备**：保持一定比例的流动资金

---

### 4.3 利润分配方案

#### 4.3.1 利润分配原则

| 原则 | 说明 |
|------|------|
| **依法分配** | 按《公司法》和章程规定分配 |
| **兼顾各方** | 兼顾公司发展与股东回报 |
| **同股同利** | 同一股份享有同等利润分配权 |
| **保护债权人** | 分配前需偿还到期债务 |

#### 4.3.2 利润分配顺序

```
1. 弥补以前年度亏损
2. 提取法定公积金（10%，累计达注册资本50%可不再提取）
3. 提取任意公积金（按股东大会决议）
4. 支付优先股股息（如发行优先股）
5. 支付普通股股利
```

#### 4.3.3 利润分配方案制定

| 步骤 | 内容 | 责任主体 |
|------|------|----------|
| 1. 财务核算 | 年度利润计算 | 财务部 |
| 2. 方案拟订 | 利润分配方案草案 | 财务部 |
| 3. 董事会审议 | 审议利润分配方案 | 董事会 |
| 4. 股东大会批准 | 批准利润分配方案 | 股东大会 |
| 5. 执行分配 | 实施分配 | 财务部 |

#### 4.3.4 分红政策建议

| 发展阶段 | 分红比例建议 | 说明 |
|----------|--------------|------|
| **初创期** | 0%-10% | 利润再投入业务发展 |
| **成长期** | 10%-30% | 适度回报股东 |
| **成熟期** | 30%-50% | 稳定分红 |
| **上市前** | 按上市要求 | 符合监管规定 |

#### 4.3.5 公积金使用

| 公积金类型 | 提取比例 | 用途 |
|------------|----------|------|
| **法定公积金** | 10% | 弥补亏损、扩大生产、转增资本 |
| **任意公积金** | 由股东大会决议 | 由股东大会决定用途 |

| 3 | 公司设立方式 | 必须 |
| 4 | 公司注册资本、已发行股份数和设立时发行的股份数 | 必须 |
| 5 | 发行类别股的，每一类别股的股份数及其权利和义务 | 如适用 |
| 6 | 发起人的姓名或者名称、认购的股份数、出资方式 | 必须 |
| 7 | 董事会的组成、职权和议事规则 | 必须 |
| 8 | 公司法定代表人的产生、变更办法 | 必须 |
| 9 | 监事会的组成、职权和议事规则 | 必须 |
| 10 | 公司利润分配办法 | 必须 |
| 11 | 公司的解散事由与清算办法 | 必须 |
| 12 | 公司的通知和公告办法 | 必须 |
| 13 | 股东会认为需要规定的其他事项 | 可选 |

### 5.3 风险提示

1. **法律合规风险**：确保公司治理结构符合《公司法》及最新法规要求
2. **股权纠纷风险**：明确股权授予、成熟、退出机制，避免纠纷
3. **控制权风险**：合理设计股权结构，保护创始人控制权
4. **税务风险**：做好股权激励的税务筹划
5. **治理僵局风险**：避免股权过于分散导致决策困难

### 5.4 实施建议

1. **专业支持**：聘请专业律师、会计师参与架构设计
2. **分步实施**：根据公司发展阶段逐步完善治理结构
3. **动态调整**：根据业务发展和融资情况适时调整
4. **文档管理**：规范各类会议记录和决议文件
