# AI跨境新手加速器 - Demo测试脚本

## 项目概述
本测试脚本用于对Demo进行全面的功能测试、性能测试和安全测试。

## 测试环境
- 后端框架: NestJS
- 前端框架: Next.js
- 数据库: PostgreSQL
- 测试工具: Jest + Supertest + Artillery

## 测试覆盖范围

### 1. 用户模块测试
- [ ] 注册功能测试
- [ ] 登录功能测试
- [ ] Token验证测试
- [ ] 权限控制测试

### 2. AI选品模块测试
- [ ] 输入验证测试
- [ ] 结果返回测试
- [ ] 错误处理测试
- [ ] 响应时间测试

### 3. 会员模块测试
- [ ] 会员信息查询测试
- [ ] 会员等级判断测试
- [ ] 权益控制测试

### 4. 安全模块测试
- [ ] SQL注入防护测试
- [ ] XSS防护测试
- [ ] 接口限流测试
- [ ] 未授权访问测试

### 5. 性能测试
- [ ] 页面加载时间测试
- [ ] API响应时间测试
- [ ] 并发测试

## 运行测试

### 单元测试
```bash
cd ./项目/backend
npm test
```

### API集成测试
```bash
npm run test:e2e
```

### 性能测试
```bash
artillery run performance-test.yml
```
