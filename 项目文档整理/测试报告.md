## 五、修复建议

### 5.1 紧急修复 (P0)

#### 1. JWT密钥安全问题
```
文件: ./项目/backend/src/app.module.ts
问题: JWT_SECRET硬编码默认值
建议: 确保.env文件中JWT_SECRET为强随机密钥
```

#### 2. 登录防暴力破解
```typescript
// 在UserController中添加限流
import { ThrottlerGuard, ThrottlerModule } from '@nestjs/throttler';

@UseGuards(ThrottlerGuard)
@Controller('auth')
export class UserController {
  // ... 登录接口
}
```

#### 3. 资源隔离验证
```typescript
// 在ProductService.findOne中添加用户验证
async findOne(id: string, userId: string) {
  const product = await this.prisma.product.findUnique({ where: { id } });
  if (product && product.userId !== userId) {
    throw new ForbiddenException('无权访问该资源');
  }
  return product;
}
```

#### 1. 添加Helmet安全头
```typescript
// main.ts
```

#### 2. 完善错误处理
```typescript
// 统一错误响应格式
{
```
