### 【第一部分】入驻前的准备工作

#### 1.1 先想清楚这四个问题

在做入驻准备之前，你需要先想清楚四个核心问题：

**问题一：我要做跨境店还是本土店？**

这是最关键的选择，决定了你后续运营的整个模式。

**跨境店（Cross-border）**：在中国内地或香港注册，货物从中国直接发往海外消费者。

**本土店（Local）**：在目标市场国家注册，货物先运到当地的仓库，再从当地发货给消费者。

两者各有优劣，我会在后面详细展开。

**问题二：我有注册什么类型的公司？**

TikTok Shop对不同类型的公司开放不同的店铺类型：

| 公司类型 | 可注册店铺 | 需要的材料 |
|----------|------------|------------|
| 中国内地企业 | 跨境POP店、全托管 | 营业执照、法人身份证 |
| 中国香港企业 | 跨境POP店 | 公司注册证明、商业登记证 |
| 个体工商户 | 部分市场跨境店 | 营业执照、法人身份证 |
| 目标市场本土企业 | 本土店 | 本地公司注册文件 |

**问题三：我的货源和仓库在哪里？**

如果你做跨境店，你需要把货发到国内的TikTok官方仓库或者第三方仓库。

如果你做本土店，你需要先把货发到目标市场的海外仓。

**问题四：我的启动资金有多少？**

不同店铺类型、不同的市场，启动资金差异很大：

- 马来西亚跨境店：最低3,000-5,000元可启动
- 泰国跨境店：最低5,000-8,000元可启动
- 本土店：最低需要准备15,000-30,000元（含海外仓费用）

#### 1.2 入驻材料清单

不管你选择哪种店铺类型，以下材料是必备的：

**基础材料（必须）：**
1. **营业执照**  
   - 中国内地企业：需要清晰的公司营业执照（目前支持有限公司、个人独资企业、合伙企业、个体工商户）
   - 中国香港企业：公司注册证明书（CI）+ 商业登记证（BR）

2. **法人身份证**  
   - 身份证正反面照片
   - 身份证有效期需大于6个月
   - 必须是营业执照上的法人本人

3. **联系方式**  
   - 手机号（国内手机号即可）
   - 邮箱（建议使用企业邮箱，避免后续账号找回问题）

**补充材料（根据情况）：**

| 材料 | 适用场景 | 说明 |
|------|----------|------|
| 第三方平台经验证明 | 跨境POP店 | 需要提供Shopee/Lazada/亚马逊/淘宝等平台店铺链接和截图，证明你有电商运营经验 |
| 仓库地址证明 | 部分市场 | 需要提供收货仓库的地址证明 |
| 品牌授权书 | 销售品牌商品 | 如果你销售的品牌需要授权，需要提供品牌授权文件 |
| 产品认证 | 部分品类 | 美妆、食品等需要提供目标市场的认证（如FDA、Halal等） |

**材料准备注意事项：**

**第一，营业执照要清晰。**  
拍摄或扫描时确保文字清晰可读，边角完整，没有反光。建议使用扫描仪或者"扫描全能王"等App。

**第二，法人信息要和营业执照一致。**  
如果法人已经变更，需要先完成工商变更，再申请入驻。

**第三，第三方平台经验很重要。**  
TikTok Shop目前对新手卖家有一定门槛，需要你提供在其他平台的运营经验。Shopee、Lazada、亚马逊、速卖通、淘宝天猫都可以。

### 【第二部分】店铺类型深度解析

#### 2.1 跨境POP店

**适合人群：**
- 中国卖家，没有海外公司
- 想先小成本试水
- 产品有供应链优势

**什么是跨境POP店？**

POP是"Platform Open Program"的缩写，也就是开放平台模式。简单来说，你作为商家入驻TikTok Shop平台，在平台上开店销售商品，平台帮你对接流量和用户。

**跨境POP店的特点：**

| 维度 | 说明 |
|------|------|
| 注册地 | 中国内地或香港 |
| 发货地 | 中国境内 |
| 物流模式 | 直发（7-15天）或官方仓（3-7天） |
| 回款周期 | 15-30天 |
| 平台佣金 | 1%-8%（品类不同） |

**支持的市场：**
目前TikTok Shop跨境POP店支持马来西亚、泰国、越南、菲律宾、新加坡五国。

**开通要求：**
1. 中国内地或香港公司（部分市场支持个体工商户）
2. 有第三方平台运营经验
3. 缴纳店铺保证金（根据类目，1,000-10,000元不等）

**优劣分析：**

✅ 优势：
- 无需在目标市场建立公司
- 启动成本低
- 可以从中国直接发货
- 有官方物流支持

❌ 劣势：
- 物流时效长（7-15天）
- 部分品类受限（如食品、药品）
- 退货处理麻烦
- 竞争激烈

#### 2.2 本土店

**适合人群：**
- 有海外公司资源
- 想做长线品牌
- 产品适合本地化运营

**什么是本土店？**

本土店就是在目标市场国家注册的店铺，货物需要先运到当地的仓库，再从本地发货给消费者。

**本土店的特点：**

| 维度 | 说明 |
|------|------|
| 注册地 | 目标市场国家（马来西亚/泰国等） |
| 发货地 | 目标市场本地仓库 |
| 物流模式 | 本地快递（1-3天） |
| 回款周期 | 7-15天 |
| 平台佣金 | 0.5%-5%（比跨境店低） |

**开通要求：**

| 市场 | 公司要求 | 个人要求 |
|------|----------|----------|
| 马来西亚 | 马来西亚公司注册文件 | 马来西亚身份证 |
| 泰国 | 泰国公司注册文件 | 泰国身份证/护照 |
| 越南 | 越南公司注册文件 | 越南身份证 |
| 菲律宾 | 菲律宾公司注册文件 | 菲律宾身份证 |
| 印尼 | 印尼公司注册文件 | 印尼身份证 |

**优劣分析：**

✅ 优势：
- 物流时效快（1-3天达）
- 平台佣金低
- 有流量扶持政策
- 可以参加更多活动
- 消费者信任度高

❌ 劣势：
- 需要本土公司或个人身份
- 需要建立海外仓
- 启动成本高
- 运营复杂度增加
- 资金回笼周期长

#### 2.3 全托管模式

**适合人群：**
- 工厂型卖家，有供应链优势
- 希望省心省力
- 愿意接受较低利润

**什么是全托管模式？**

全托管模式下，你只需要供货，TikTok Shop负责运营、推广、客服、履约等所有环节。你扮演的是"供应商"的角色。

**全托管的特点：**

| 维度 | 说明 |
|------|------|
| 运营主体 | TikTok Shop平台运营 |
| 定价权 | 平台定价 |
| 利润模式 | 按供货价结算 |
| 结算周期 | 月结或半月结 |
| 风险承担 | 平台承担滞销风险 |

**优劣分析：**

✅ 优势：
- 零运营压力
- 不用担心流量
- 账期稳定
- 无需处理客服

❌ 劣势：
- 利润较低（通常比自营低30%-50%）
- 没有定价权
- 产品被压价风险
- 无法建立自己的品牌

#### 2.4 店铺类型对比总结

| 对比维度 | 跨境POP店 | 本土店 | 全托管 |
|----------|----------|--------|--------|
| 启动资金 | 低（5千-2万） | 高（3万-10万） | 中（1万-5万） |
| 运营难度 | 中 | 高 | 低 |
| 物流时效 | 慢（7-15天） | 快（1-3天） | 快（1-3天） |
| 利润空间 | 中高 | 高 | 低 |
| 风险程度 | 中 | 高 | 低 |
| 适合人群 | 新手小白 | 有资源老手 | 工厂型卖家 |

**新手建议：**  
如果你完全没有经验，资金有限，我建议你从**跨境POP店**起步。跑通流程、积累经验后，再考虑本土店或全托管。

### 【第三部分】账号注册实操流程

#### 3.1 注册入口

**电脑端注册（推荐）：**
1. 打开 TikTok Shop Seller Center 官网
2. 选择目标市场（如：马来西亚）
3. 点击"注册"按钮

**手机端注册：**
1. 下载 TikTok Shop Seller App
2. 打开App，点击"注册"
3. 按提示填写信息

**官网地址：**
- 跨境店：seller-ids.tiktok.com
- 马来西亚：seller-my.tiktok.com
- 泰国：seller-th.tiktok.com
- 越南：seller-vn.tiktok.com
- 菲律宾：seller-ph.tiktok.com

#### 3.2 注册步骤详解

**第一步：选择市场**

打开注册页面后，首先选择你想入驻的市场。

**注意：**  
- 如果你想做多个市场，需要分别注册
- 同一个营业执照可以在不同市场分别注册
- 部分市场对营业执照类型有要求

**第二步：选择注册方式**

有两种注册方式：

| 方式 | 说明 |
|------|------|
| 短信验证码注册 | 使用手机号+验证码注册，适合快速入驻 |
| 邮箱注册 | 使用邮箱注册，后续可以绑定手机 |

**推荐：** 优先使用手机号注册，后续可以添加邮箱作为备用登录方式。

**第三步：选择公司主体类型**

| 类型 | 所需材料 |
|------|----------|
| 中国内地企业 | 营业执照、法人身份证 |
| 中国香港企业 | 公司注册证明、商业登记证 |
| 个体工商户 | 营业执照、法人身份证（部分市场支持） |

**第四步：填写公司信息**

| 字段 | 填写说明 |
|------|----------|
| 公司名称 | 与营业执照上的名称一致 |
| 统一社会信用代码 | 18位代码，营业执照上有 |
| 注册地址 | 营业执照上的地址 |
| 营业期限 | 如果是长期有效，选择"长期" |
| 经营范围 | 建议选择"大类目"，避免后续限制 |

**第五步：填写法人信息**

| 字段 | 填写说明 |
|------|----------|
| 法人姓名 | 与身份证一致 |
| 身份证号码 | 18位身份证号 |
| 身份证有效期 | 填写实际有效期 |
| 法人手机号 | 国内手机号即可 |
| 法人邮箱 | 建议使用常用邮箱 |

**第六步：上传证件**

需要上传：
1. 营业执照（彩色照片或扫描件）
2. 法人身份证正反面

**上传要求：**
- 文件格式：JPG/PNG/PDF
- 文件大小：不超过5MB
- 图片清晰，文字可读
- 四个边角完整显示

**第七步：填写店铺信息**

| 字段 | 填写说明 |
|------|----------|
| 店铺名称 | 建议使用品牌名+店名格式，如"XXX官方旗舰店" |
| 主营类目 | 选择你要销售的主要品类 |
| 仓库地址 | 填写你的发货仓库地址 |

**店铺名称注意事项：**
- 2-30个字符
- 不能使用他人品牌名
- 不能使用TikTok官方相关词汇（如TikTok、TK等）
- 不能使用敏感词（如"旗舰"、"官方"等）

**第八步：提交审核**

完成以上步骤后，点击"提交"。

**审核时间：**
- 正常情况下：1-3个工作日
- 高峰期（如大促前）：可能延长至5-7个工作日

**审核结果：**
- 通过：你会收到邮件和短信通知，可以开始上架商品
- 驳回：会收到驳回原因，按要求修改后重新提交

#### 3.3 常见驳回原因及解决方案

| 驳回原因 | 解决方案 |
|----------|----------|
| 营业执照不清晰 | 重新拍摄或扫描，确保四个边角完整、文字清晰 |
| 法人信息不一致 | 检查营业执照和身份证是否一致，联系平台客服 |
| 店铺名称违规 | 修改店铺名称，去除敏感词和品牌词 |
| 缺少第三方平台经验 | 提供其他电商平台的店铺链接和截图 |
| 营业执照过期 | 需要更新营业执照，或联系平台说明情况 |

### 【第四部分】新手激励政策与任务

#### 4.1 平台新手激励政策

TikTok Shop为了吸引新卖家入驻，会不定期推出新手激励计划。以下是常见的新手权益：

**新手期权益（入驻后30天内）：**
1. 免佣金期：部分品类享受15-30天免佣
2. 流量扶持：新店曝光加权
3. 专属小二：对接平台运营指导
4. 活动优先权：优先参与平台活动

**新手任务（入驻后90天内完成）：**

| 任务 | 完成条件 | 奖励 |
|------|----------|------|
| 完成首单 | 入驻后30天内出首单 | 获得"新店标识" |
| 达到GMV门槛 | 90天内GMV达到$500 | 解锁更多活动资格 |
| 上传商品数 | 上传至少50个商品 | 获得选品工具高级权限 |
| 完成新手考试 | 通过TikTok Shop新手考试 | 获得运营指导资料 |

**如何查看激励政策：**
1. 登录TikTok Shop Seller Center
2. 进入"权益中心"或"成长中心"
3. 查看当前可参与的活动和任务

#### 4.2 入驻后必做事项

完成注册后，你需要立即完成以下事项：

**第一，完善店铺基础设置**

| 设置项 | 操作 |
|--------|------|
| 店铺Logo | 上传清晰的店铺Logo |
| 店铺简介 | 填写店铺介绍，便于用户了解 |
| 联系方式 | 设置客服联系方式 |
| 退货地址 | 设置国内退货仓库地址 |
| 发货地址 | 设置实际发货仓库地址 |

**第二，完成支付账户绑定**

| 步骤 | 操作 |
|------|------|
| 2.1 | 进入"账户设置" |
| 2.2 | 选择"支付账户" |
| 2.3 | 添加收款账户（PingPong、Payoneer、WorldFirst等） |
| 2.4 | 验证账户信息 |

**第三，设置运费模板**

| 步骤 | 操作 |
|------|------|
| 3.1 | 进入"物流设置" |
| 3.2 | 创建运费模板 |
| 3.3 | 设置包邮条件 |
| 3.4 | 设置偏远地区附加费 |

**第四，完成新手考试**

| 步骤 | 操作 |
|------|------|
| 4.1 | 进入"学习中心" |
| 4.2 | 找到"新手必修"课程 |
| 4.3 | 完成所有视频学习 |
| 4.4 | 参加新手考试（80分及格） |

### 【课程总结】

## 三、核心案例

### 案例一：从0到100单的跨境卖家

**背景：**
卖家阿杰，深圳人，2024年3月开始做TikTok Shop，选择马来西亚跨境店，主做美妆工具。

**入驻经历：**
1. 第一天：准备材料（营业执照、法人身份证、Shopee店铺截图）
2. 第三天：完成注册提交
3. 第五天：审核通过
4. 第七天：完成店铺基础设置
5. 第十天：上架第一批商品（15个）
6. 第二十天：完成首单
7. 第四十天：日均订单达到10单

**关键动作：**
1. 选择自己熟悉的品类（美妆），有Shopee经验可复用
2. 没有急于上品，先把店铺基础打牢
3. 认真完成新手任务，获得平台流量扶持
4. 第一个月重点优化商品主图和标题

**数据表现（第一个月）：**
- 总订单：156单
- GMV：$2,340
- 利润率：38%
- 退货率：2.5%

**启示：** 注册只是第一步，店铺基础设置同样重要。

### 案例二：本土店注册踩坑记

**背景：**
卖家小雪，义乌人，2024年初想做泰国本土店，找了代理注册泰国公司。

**踩坑经历：**
1. **第一个坑：代理不靠谱**  
   找的代理收了3万元注册费，结果3个月都没完成公司注册，最后发现代理是骗子。

2. **第二个坑：注册文件不符合要求**  
   好不容易找到第二家代理，用了2个月完成公司注册，但银行开户时发现文件格式不符合要求。

3. **第三个坑：海外仓选址失误**  
   公司注册完成后，开始建海外仓。选择在曼谷市中心租了一个仓库，月租2万泰铢（人民币约4,000元），后来发现位置太偏，物流时效反而变慢。

**最终结果：**
折腾了6个月，总投入超过15万元，最终选择放弃泰国本土店，转做跨境店。

**教训总结：**
1. 选择代理要谨慎，查看资质、签合同、分期付款
2. 提前了解目标市场的公司注册要求
3. 海外仓选址要考虑物流网络，而不是单纯看租金
4. 新手不建议直接做本土店，风险太大

**启示：** 本土店看起来美好，但背后的复杂度和成本远超想象。

## 四、实操步骤清单

### 步骤一：材料准备（1-2小时）

```
□ 1.1 确认公司类型（内地企业/香港企业/个体工商户）
□ 1.2 准备营业执照原件和复印件
□ 1.3 准备法人身份证正反面照片
□ 1.4 准备第三方平台店铺截图（Shopee/Lazada/淘宝等）
□ 1.5 确认仓库地址
□ 1.6 检查所有材料是否清晰、有效
```

### 步骤二：账号注册（30分钟-1小时）

```
□ 2.1 打开TikTok Shop Seller Center
□ 2.2 选择目标市场
□ 2.3 选择注册方式（手机号/邮箱）
□ 2.4 选择公司主体类型
□ 2.5 填写公司信息
□ 2.6 填写法人信息
□ 2.7 上传证件材料
□ 2.8 填写店铺信息
□ 2.9 提交审核
```

### 步骤三：入驻后基础设置（2-3小时）

```
□ 3.1 登录店铺，完善店铺Logo
□ 3.2 填写店铺简介
□ 3.3 设置联系方式
□ 3.4 绑定收款账户（PingPong/Payoneer等）
□ 3.5 设置发货地址
□ 3.6 设置退货地址
□ 3.7 创建运费模板
□ 3.8 完成新手考试
□ 3.9 上传至少10个商品
```

### 步骤四：新手任务完成（7天内）

```
□ 4.1 完成首单（30天内）
□ 4.2 店铺设置完整度100%
□ 4.3 绑定收款账户并验证
□ 4.4 设置至少3个运费模板
□ 4.5 上传至少50个商品
□ 4.6 参加新手培训并通过考试
```
