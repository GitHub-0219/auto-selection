### 【第一部分】选品的本质

#### 1.1 选品≠选产品，而是选"需求"

很多新手卖家对选品的理解是：从1688上找一些便宜的产品，放到TikTok Shop上去卖。

这种理解，**完全搞反了**。

选品的起点不是产品，而是**需求**。

**正确的选品逻辑应该是这样的：**

```
第一步：发现需求
   ↓
   东南亚的年轻妈妈有什么痛点？
   泰国的上班族中午吃什么？
   马来西亚的穆斯林女性需要什么样的护肤品？
   
第二步：寻找解决方案
   ↓
   调研市场上有没有解决这些痛点的产品？
   这些产品有什么不足？
   我能提供更好的解决方案吗？
   
第三步：验证市场需求
   ↓
   市场上类似产品卖得好不好？
   销量趋势是上升还是下降？
   竞争激烈程度如何？
   
第四步：确定产品
   ↓
   基于需求验证结果，确定最终的产品
```

**举例说明：**

假设你发现：东南亚很多年轻妈妈一个人带娃，每天忙得没时间好好吃饭。

这是需求。

然后你调研发现：市场上有很多"懒人厨房"类产品，比如多功能切菜器、快速烹饪工具等。

但现有产品有什么不足？要么太大不方便，要么功能太单一，要么在TikTok上的展示效果不好。

然后你验证：多功能切菜器在TikTok上的视频播放量很高，很多视频都是几十万播放，说明用户对这个产品感兴趣。

最后，你确定了产品：选择一款小巧多功能、展示效果好的切菜器。

**这就是选品的完整链路：需求→解决方案→验证→产品。**

#### 1.2 选品的三层价值

一个好的产品，应该同时具备三层价值：

**第一层：使用价值（Functional Value）**

产品能解决用户的实际问题。

| 维度 | 说明 | 举例 |
|------|------|------|
| 实用性 | 产品能实际使用 | 切菜器能切菜 |
| 效率提升 | 让事情更容易/更快 | 3秒切完一个土豆 |
| 成本节省 | 省钱/省时间 | 比外卖便宜，比自己做方便 |

**第二层：情绪价值（Emotional Value）**

产品能满足用户的情感需求。

| 维度 | 说明 | 举例 |
|------|------|------|
| 成就感 | 用完有满足感 | 做出了一顿大餐 |
| 归属感 | 符合某个群体认同 | 马来西亚穆斯林专属 |
| 安全感 | 让人放心 | 食品级材质，安全无毒 |

**第三层：社交价值（Social Value）**

产品能被传播、被展示。

| 维度 | 说明 | 举例 |
|------|------|------|
| 可分享性 | 值得发TikTok分享 | 开箱视频很有看点 |
| 话题性 | 容易引发讨论 | 奇葩设计引发吐槽 |
| 展示效果 | 在镜头前好看 | 颜色鲜艳，视觉冲击 |

**在TikTok上，这三层价值缺一不可。**

因为TikTok是内容电商，用户首先是被内容打动，然后才购买产品。

如果产品没有展示效果，在TikTok上就很难火起来。

所以，在选品时，你需要同时问自己三个问题：

```
这个产品能解决用户的实际问题吗？  → 使用价值
这个产品能让用户感觉好吗？          → 情绪价值
这个产品在TikTok上好看吗？能让人想分享吗？  → 社交价值
```

#### 1.3 选品的底层逻辑

选品的底层逻辑可以总结为一个公式：

**成功选品 = 需求真实 × 竞争可控 × 供应链稳定 × 内容可展示 × 合规无风险**

| 要素 | 含义 | 判断标准 |
|------|------|----------|
| 需求真实 | 市场确实有人需要这个产品 | 有搜索量、有销量 |
| 竞争可控 | 不是红海市场，有差异化机会 | 不是几百家店卖同款 |
| 供应链稳定 | 能稳定供货，不会断货 | 供应商可靠，价格稳定 |
| 内容可展示 | 在TikTok上能拍出好看的内容 | 有展示亮点，适合视频 |
| 合规无风险 | 符合当地法规，不会有违规风险 | 不需要特殊资质认证 |

**这五个要素缺一不可。**

如果只满足前两个，但供应链不稳定，可能卖着卖着就断货了。

如果前四个都满足，但产品需要Halal认证而你没有，轻则下架，重则封店。

所以，选品是一个需要综合考量的系统性决策，不是简单地看哪个产品便宜就选哪个。

### 【第二部分】选品的核心原则

#### 2.1 原则一：市场需求验证优先

**选品的第一个原则：市场需求验证优先。**

很多新手卖家选品的逻辑是：我觉得这个产品好卖，所以我要卖。

这种逻辑是错的。

**正确的逻辑应该是：市场验证了这个产品好卖，所以我才卖。**

怎么验证市场需求？

**方法一：看数据**

| 数据来源 | 查看内容 | 判断标准 |
|----------|----------|----------|
| TikTok选品广场 | 热卖榜单 | 月销量>500的产品有参考价值 |
| 竞品店铺 | 竞品销量 | 销量稳定且持续的产品 |
| 谷歌趋势 | 搜索热度 | 热度持续上升的产品 |
| 第三方工具 | 市场数据分析 | 多维度验证 |

**方法二：看趋势**

| 趋势类型 | 说明 | 案例 |
|----------|------|------|
| 季节性趋势 | 节日、天气带来的短期需求 | 防晒霜（夏季）、雨伞（雨季） |
| 长期趋势 | 消费升级、人口结构变化带来的长期需求 | 健康类产品、智能家居 |
| 热点趋势 | 社会热点事件带动的短期需求 | 某个电影带火的周边产品 |

**方法三：看竞品**

| 分析维度 | 查看内容 |
|----------|----------|
| 竞品数量 | 搜索关键词，看有多少卖家 |
| 竞品销量 | 看头部卖家月销量多少 |
| 竞品评价 | 看差评主要是什么问题 |
| 竞品定价 | 市场价格带在什么区间 |

**判断标准：**
- 月销量>1000：市场需求旺盛
- 月销量500-1000：市场需求中等
- 月销量<500：市场需求较小，谨慎选择

#### 2.2 原则二：供应链可控性

**选品的第二个原则：供应链可控性。**

产品选好了，但供应链不稳定，一切都白搭。

供应链可控性包括三个维度：

**维度一：供应商稳定性**

| 评估项 | 问题清单 |
|--------|----------|
| 供货能力 | 供应商月产能多少？能稳定供货吗？ |
| 响应速度 | 下单后多久能发货？紧急加单能处理吗？ |
| 质量稳定 | 产品质量稳定吗？批次之间差异大吗？ |
| 合作关系 | 能建立长期稳定合作吗？ |

**维度二：成本可控性**

| 评估项 | 问题清单 |
|--------|----------|
| 产品成本 | 拿货价合理吗？有没有议价空间？ |
| 物流成本 | 到仓库/到海外仓的成本多少？ |
| 其他成本 | 包装、质检、仓储等附加成本 |
| 利润空间 | 扣除所有成本后还有利润吗？ |

**维度三：备货可控性**

| 评估项 | 问题清单 |
|--------|----------|
| 补货周期 | 从下单到收货需要多少天？ |
| 安全库存 | 需要备多少安全库存？ |
| 滞销风险 | 卖不动怎么办？能退货/清货吗？ |
| 爆款应对 | 突然爆了能快速补货吗？ |

**新手建议：**

刚开始做，建议选择供应链在国内的产品，物流稳定，补货快。

不要一上来就做需要海外仓周转的产品，库存管理难度太大。

#### 2.3 原则三：物流适配性

**选品的第三个原则：物流适配性。**

TikTok Shop的发货模式决定了物流适配性的重要性。

**跨境店物流模式对比：**

| 模式 | 时效 | 成本 | 适合品类 |
|------|------|------|----------|
| 直邮小包 | 7-15天 | 低 | 小件、低价值商品 |
| 官方仓 | 3-7天 | 中 | 中等件、中等价值 |
| 海外仓 | 1-3天 | 高 | 大件、高价值、高复购 |

**物流适配性评估：**

| 评估项 | 判断标准 |
|--------|----------|
| 尺寸重量 | 单件重量<500g最理想，物流成本低 |
| 易碎程度 | 易碎品物流风险高，不建议新手做 |
| 保质期 | 有保质期的商品有滞销风险 |
| 体积 | 体积过大物流成本高 |
| 退货率 | 体积大的商品退货物流成本高 |

**建议新手选择的产品特征：**
```
✅ 单件重量在100g-500g之间
✅ 不易碎、不易变形
✅ 没有保质期或保质期长（>1年）
✅ 体积小，方便打包
✅ 不需要特殊存储条件
```

#### 2.4 原则四：内容可展示性

**选品的第四个原则：内容可展示性。**

这是TikTok Shop选品和传统电商最大的区别。

**传统电商选品**：看转化率、好评率、复购率。

**TikTok Shop选品**：除了看这些，还要看**展示效果**。

什么是好的展示效果？

**特征一：视觉冲击力强**

| 特点 | 案例 |
|------|------|
| 颜色鲜艳 | 彩色眼影盘比透明指甲油更好拍 |
| 形状独特 | 特殊造型的厨房工具更有看点 |
| 效果明显 | 清洁类产品（前/后对比） |

**特征二：功能可演示**

| 特点 | 案例 |
|------|------|
| 操作过程可见 | 切菜器3秒切完一个土豆 |
| 效果可量化 | 吸尘器吸出一大堆灰尘 |
| 前后对比明显 | 美容仪使用前后皮肤状态对比 |

**特征三：使用场景真实**

| 特点 | 案例 |
|------|------|
| 贴近生活 | 展示每天早上化妆的场景 |
| 有代入感 | 展示一个人带娃做饭的痛点 |
| 引发共鸣 | "这个切菜器解决了我10年的烦恼" |

**特征四：引发互动**

| 特点 | 案例 |
|------|------|
| 有槽点 | 让人忍不住吐槽的设计 |
| 有惊喜 | 打开包装的那一刻让人"哇" |
| 可二创 | 用户愿意拍自己的视频 |

**怎么判断产品的展示效果？**

1. 在TikTok上搜索相关关键词，看同类产品的视频播放量
2. 如果播放量普遍较高（>10万），说明展示效果好
3. 如果播放量普遍较低（<1万），说明展示效果一般

#### 2.5 原则五：合规风险可控

**选品的第五个原则：合规风险可控。**

东南亚各国对进口商品有不同的法规要求，如果不注意，轻则下架，重则封店。

**主要合规风险：**

| 风险类型 | 说明 | 高风险品类 |
|----------|------|------------|
| 认证要求 | 部分品类需要特殊认证 | 美妆、食品、医疗器械 |
| 进口限制 | 部分商品禁止或限制进口 | 武器、电子烟等 |
| 知识产权 | 未经授权销售品牌产品 | 大牌仿品、授权商品 |
| 内容合规 | 视频/图文内容违规 | 虚假宣传、敏感内容 |

**各国特殊要求：**

| 国家 | 特殊要求 | 注意事项 |
|------|----------|----------|
| 马来西亚 | Halal认证 | 穆斯林相关品类建议获取 |
| 印尼 | BPOM认证 | 美妆、食品必须认证 |
| 泰国 | FDA认证 | 食品、药品、化妆品需要 |
| 越南 | 暂无特殊要求 | 相对宽松 |
| 菲律宾 | 暂无特殊要求 | 相对宽松 |

**新手建议：**

第一，**优先选择不需要特殊认证的品类**。美妆、食品、医疗器械新手不建议做。

第二，**不要卖任何仿牌、假货**。这是底线，一旦被发现直接封店。

第三，**了解目标市场的禁忌**。穆斯林市场不要出现猪肉相关元素，佛教国家不要有不敬佛教的内容。

### 【第三部分】选品的常见误区

#### 3.1 误区一：只看利润，不看市场需求

**典型表现：**
- "这个产品利润有50%，肯定能赚钱"
- "成本才5块钱，卖15块，利润空间大大的"
- 选了一个利润很高但几乎没人买的产品

**后果：**
产品卖不动，库存积压，资金周转困难。

**正确做法：**
先验证市场需求，再计算利润空间。市场需求是前提，利润空间是保障。

**判断标准：**
```
市场需求验证（已有销量数据）+ 利润空间合理
= 值得做

市场需求未验证（没有任何数据支撑）+ 利润空间高
= 风险极大，谨慎
```

#### 3.2 误区二：照搬国内爆款

**典型表现：**
- "这个产品在国内抖音上爆了，东南亚肯定也行"
- 不考虑东南亚的实际情况，直接复制国内选品
- 结果水土不服，卖不动

**后果：**
产品不符合目标市场需求，转化率极低。

**原因分析：**
1. 消费能力差异：东南亚人均收入比中国低，不能照搬中高端产品
2. 文化差异：东南亚有穆斯林、佛教等不同信仰，与中国差异大
3. 使用场景差异：气候、生活习惯不同，同样的产品使用场景不同
4. 审美差异：颜色偏好、设计偏好可能完全不同

**正确做法：**
选品前先了解目标市场的特殊性和消费者偏好，而不是盲目照搬。

#### 3.3 误区三：忽视本地化需求

**典型表现：**
- 把国内的产品直接翻译成英文/当地语言就上架
- 不考虑当地的使用习惯、尺寸标准、电压标准
- 买家收到货后发现不符合当地情况

**后果：**
退货率高、差评多、口碑受损。

**常见本地化问题：**

| 问题类型 | 错误做法 | 正确做法 |
|----------|----------|----------|
| 语言 | 机器翻译，语句不通 | 本地人校对了再上架 |
| 尺寸 | 用中国尺码 | 使用国际尺码或当地习惯 |
| 电压 | 220V电器卖到110V市场 | 确认电压匹配 |
| 天气 | 卖羽绒服到热带 | 考虑当地气候 |
| 节日 | 卖圣诞装饰到穆斯林国家 | 考虑当地节日 |

**正确做法：**
选品时要考虑产品与目标市场的本地化适配程度。

#### 3.4 误区四：过度依赖价格战

**典型表现：**
- "我价格比竞争对手低1块，肯定能卖过他们"
- 定价只考虑价格，不考虑产品质量和运营成本
- 陷入低价竞争，最终亏本离场

**后果：**
价格战没有赢家，利润被压垮，服务质量下降，最后两败俱伤。

**价格战的本质：**
价格战是**最笨的竞争方式**。因为价格没有最低，只有更低。

**正确做法：**
通过差异化（产品差异化、内容差异化、服务差异化）来竞争，而不是单纯的价格战。

**案例对比：**

| 策略 | 结果 |
|------|------|
| 单纯价格战 | 利润极低，一旦涨价客户就流失 |
| 差异化竞争 | 利润合理，客户忠诚度高 |

### 【第四部分】建立选品框架

#### 4.1 选品自检清单

每次选品时，用这个清单过一遍：

```
【市场需求】
□ 有明确的用户痛点/需求场景
□ 在TikTok/TikTok Shop上有类似产品销售
□ 月销量>500（验证市场需求）
□ 销量趋势上升（不是下降趋势）

【竞争分析】
□ 同款商品卖家数量<50
□ 价格带有空白区间（可以做差异化）
□ 头部卖家不是绝对垄断
□ 有差异化机会

【供应链】
□ 供应商稳定，能长期供货
□ 产品成本可控，有利润空间
□ 补货周期<15天
□ 质量稳定，批次差异小

【物流适配】
□ 单件重量<500g
□ 不易碎、不易变形
□ 无保质期或保质期长
□ 体积小，方便打包

【内容展示】
□ 视觉冲击力强
□ 功能可演示
□ 使用场景真实
□ 有展示亮点

【合规风险】
□ 不需要特殊认证
□ 没有进口限制
□ 没有知识产权风险
□ 符合当地文化和法规
```

如果以上每一项都通过，选品成功概率大大提高。

## 三、核心案例

### 案例一：成功的需求驱动选品

**背景：**
卖家阿琳，2024年初做TikTok Shop马来西亚市场，主做家居收纳。

**选品思路：**
1. **发现需求**：在TikTok上刷到很多马来西亚用户抱怨"衣柜太小，衣服太多不知道怎么收纳"
2. **验证需求**：搜索发现收纳类视频播放量普遍很高（20万+），且同类产品月销量>2000
3. **寻找差异化**：现有产品多为塑料材质，她选择了"布艺收纳袋"，颜色更温馨，更适合马来西亚家庭风格
4. **验证供应**：找到广州工厂，能提供多种尺寸，OEM起订量低

**执行结果：**
- 第一个月销量：800+
- 爆款单品：衣柜分层收纳袋（单月3000+件）
- 利润率：42%

**成功因素：**
1. 从真实需求出发，不是从产品出发
2. 做了差异化（布艺 vs 塑料）
3. 符合本地化审美
4. 供应链稳定

---

### 案例二：失败的产品驱动选品

**背景：**
卖家阿强，2024年3月做TikTok Shop越南市场。

**选品思路：**
1. 在1688上看到一款"自动雨伞收纳套"，觉得很新颖
2. 成本2.5元，定价8.9美元，利润看起来很高
3. 直接下单500件，发到越南仓库

**结果：**
- 第一周销量：5件
- 问题：越南用户不理解这个产品有什么用（需求不成立）
- 第二周：0销量
- 最后清仓处理，亏损3000元

**失败原因：**
1. 没有验证市场需求，纯粹是"我觉得好卖"
2. 没有考虑越南用户的生活场景
3. 东南亚雨伞使用频率不高，这个产品不实用

**教训：**
先验证需求，再考虑利润。不要因为"便宜+利润高"就冲动选品。
