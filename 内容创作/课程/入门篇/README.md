## 课程内容概览

### 第1课：TikTok Shop东南亚市场概览

**课程时长**：约25分钟  
**核心内容**：
- TikTok Shop商业模式解析
- 东南亚五国市场深度分析
- 新手市场选择策略

**亮点章节**：
- 内容电商 vs 货架电商的本质区别
- 马来西亚/泰国/越南/菲律宾/印尼五国市场详解
- 新手首选市场推荐与多市场布局策略

**实战任务**：
- 完成五国市场调研报告
- 确定首批入驻市场

### 第2课：平台入驻完整流程

**课程时长**：约30分钟  
**核心内容**：
- 入驻前的四个核心问题
- 三种店铺类型详解（跨境POP店/本土店/全托管）
- 完整注册流程实操
- 新手激励政策

**亮点章节**：
- 跨境店 vs 本土店 vs 全托管模式深度对比
- 常见驳回原因及解决方案
- 入驻后必做事项清单

**实战任务**：
- 完成入驻材料准备
- 完成账号注册
- 完成新手任务

- 数据与财务模块

**亮点章节**：

## 内容规格

### 每课标准内容

每课包含以下完整内容：

| 内容模块 | 说明 |
|----------|------|
| 逐字稿 | 完整讲课内容，约3000-5000字，25-35分钟 |
| PPT大纲 | 15-18页PPT结构设计 |
| 核心案例 | 2-3个真实或典型案例 |
| 实操清单 | 分步骤操作指南 |
| 延伸资料 | 推荐阅读、工具、数据源 |

### 质量标准

- ✅ 内容准确：数据和案例真实可验证
- ✅ 语言通俗：新手友好，避免专业术语堆砌
- ✅ 结构清晰：有逻辑、有条理、有层次
- ✅ 实操性强：看完就能上手操作
