## 附录：技术规格

### 视频参数
| 项目 | 规格 |
|-----|------|
| 分辨率 | 1080 x 1920 (9:16竖屏) |
| 帧率 | 30fps |
| 时长 | 2分45秒 |
| 格式 | MP4 |
| 音频 | AAC, 128kbps |

### 背景音乐建议
- **片头**：科技感Intro音乐，5s
- **主体**：轻快的背景BGM，持续全程，音量控制在-20dB
- **片尾**：Outro音乐，渐隐收尾

### 配音要求
- **声音风格**：男声，成熟稳重，专业可信
- **语速**：约200字/分钟
- **语调**：自然流畅，重点词适当加重

### 动画风格
- 保持与UI设计一致的风格
- 转场使用滑入/渐变，避免硬切
- 数据图表使用生长动画
- 按钮交互有明显反馈
