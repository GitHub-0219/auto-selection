### 6.2 操作步骤详解

#### Step 1: 注册与配置
```
1. 打开 heygen.com
2. 点击右上角 "Sign Up"
3. 选择注册方式（推荐Google账号）
4. 填写基本信息
5. 选择 "Creator" 套餐（¥449/月）
6. 完成支付（支持信用卡）
```

#### Step 2: 选择数字人
```
1. 进入控制台，点击 "Create Video with AI"
2. 点击左侧 "Avatar" 选项
3. 使用筛选器：
   - Language: Chinese
   - Gender: 根据品牌调性选择
   - Style: Business/Casual
4. 选择合适的数字人，点击使用
5. 可点击 "Customize" 调整服装/发型
```

#### Step 3: 输入脚本
```
1. 在文本框中输入完整脚本
2. 点击语言选择，选择 "Chinese (Mandarin)"
3. 调整语速滑块（建议1.0-1.1）
4. 点击 "Preview" 预览效果
5. 如有不满意，修改脚本后重新预览
```

#### Step 4: 添加背景与素材
```
1. 点击 "Background" 添加背景
   - 可选择内置背景
   - 或上传自定义背景图（建议1920x1080）
2. 点击 "Elements" 添加素材
   - Logo（建议PNG透明背景）
   - 图片/视频片段
3. 点击 "Captions" 添加字幕
```

#### Step 5: 生成与下载
```
1. 确认所有设置无误
2. 点击右下角 "Generate"
3. 等待处理（可在 "My Videos" 查看进度）
4. 处理完成后，点击下载按钮
5. 选择分辨率（推荐1080P）
6. 下载完成
```
