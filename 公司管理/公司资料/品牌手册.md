# 品牌手册

## 一、品牌概述

### 1.1 品牌定义
品牌是科技管理集团传递给市场的整体形象和价值主张，是客户认知和情感连接的载体。

### 1.2 品牌架构
| 层级 | 内容 |
|-----|-----|
| 母品牌 | 科技管理集团（TechVision Group） |
| 产品品牌 | Auto选品 |
| 未来产品线 | 待定 |

---

## 二、品牌定位

### 2.1 品牌定位陈述
**为谁**：年销售额50万-5000万的跨境电商卖家

**解决什么问题**：帮助他们降低选品门槛，提高选品成功率

**核心价值**：AI智能、数据驱动、简单高效

**差异化**：更精准的AI算法 + 更懂卖家的产品体验

### 2.2 品牌定位三角
| 维度 | 内容 |
|-----|-----|
| 我们是（身份） | 跨境电商AI选品领域的创新者 |
| 我们做（价值） | 用AI技术赋能卖家，降低选品风险 |
| 我们服务（对象） | 跨境电商卖家，尤其是中小型卖家 |

### 2.3 品牌个性
| 个性维度 | 描述 |
|---------|-----|
| 专业 | 专业技术，值得信赖 |
| 创新 | 敢于突破，不断创新 |
| 务实 | 脚踏实地，注重效果 |
| 友好 | 易于使用，亲切友好 |

## 五、品牌视觉识别系统（VI）

### 5.1 Logo
**设计原则**：
- 简洁醒目，易于识别
- 体现科技感和数据感
- 具有国际化视野

**Logo组成**：
- 主标识：TechVision / TVG
- 产品标识：Auto选品
- 组合方式：横排/竖排

**使用规范**：
- 最小使用尺寸：不低于20mm
- 安全空间：Logo周围保留适当空白
- 禁止事项：不得拉伸、变形、变色

### 5.2 品牌色彩
| 色彩类型 | 色值 | 使用场景 |
|---------|-----|---------|
| 主色-科技蓝 | #2563EB | Logo、主要按钮、重点强调 |
| 辅助蓝 | #3B82F6 | 次要元素、背景色 |
| 强调色-数据绿 | #10B981 | 增长、正向数据、成功状态 |
| 警示色-橙色 | #F59E0B | 警告、注意 |
| 错误色-红色 | #EF4444 | 错误、危险 |
| 深色背景 | #0F172A | 深色模式背景 |
| 浅色背景 | #F8FAFC | 浅色模式背景 |
| 正文文字 | #1E293B | 主要文字 |
| 次要文字 | #64748B | 辅助说明文字 |

### 5.3 品牌色彩应用
| 应用场景 | 推荐色彩 |
|---------|---------|
| 网站主色调 | 科技蓝 + 白色 |
| 按钮/CTA | 科技蓝 |
| 数据图表 | 蓝色系 + 数据绿 |
| 空状态 | 灰色调 |
| 成功反馈 | 数据绿 |

### 5.4 字体规范
| 字体类型 | 字体选择 | 使用场景 |
|---------|---------|---------|
| 中文字体 | 思源黑体/阿里巴巴普惠体 | 正文、标题 |
| 英文字体 | Inter/SF Pro Display | 正文、标题 |
| 数字字体 | DM Sans/数字专用 | 数据展示 |
| 代码字体 | JetBrains Mono/Fira Code | 代码展示 |

**字号规范**：
| 元素 | PC端字号 | 移动端字号 |
|-----|---------|-----------|
| H1 标题 | 32px | 28px |
| H2 标题 | 24px | 22px |
| H3 标题 | 20px | 18px |
| 正文 | 16px | 15px |
| 辅助文字 | 14px | 13px |
| 注释文字 | 12px | 12px |

### 5.5 图标规范
- **风格**：线性图标，简洁现代
- **线宽**：统一线宽（2px）
- **圆角**：统一圆角（4px）
- **颜色**：跟随文字颜色或品牌色
- **使用**：优先使用系统图标库

### 5.6 插图风格
- **风格**：扁平插画、数据可视化风格
- **配色**：使用品牌色彩
- **场景**：用于空状态、功能引导、品牌宣传

### 5.7 品牌图形元素
| 元素 | 说明 | 使用场景 |
|-----|-----|---------|
| 几何图形 | 圆形、方形、三角形的组合 | 背景装饰 |
| 数据元素 | 图表、图形、数据点 | 体现数据能力 |
| AI元素 | 神经元、网络节点 | 体现AI能力 |

## 七、品牌话术规范

### 7.1 品牌描述
**一句话介绍**：
> 科技管理集团是一家专注跨境电商AI选品服务的科技公司，让全球贸易更简单。

**详细描述**：
> 科技管理集团（TechVision Group）是一家专注于跨境电商AI选品及智能运营服务的科技公司。我们的核心产品Auto选品基于大数据和AI算法，帮助跨境电商卖家快速发现蓝海市场，降低选品风险，实现业务增长。公司以"让全球贸易更简单"为愿景，致力于用科技赋能卖家，用智能驱动增长。

### 7.2 品牌Slogan
| 版本 | 标语 | 使用场景 |
|-----|-----|---------|
| 主Slogan | 让选品更简单，让增长更确定 | 品牌宣传 |
| 产品Slogan | AI驱动，智慧选品 | 产品推广 |
| 价值Slogan | 数据洞察，精准决策 | 功能宣传 |

### 7.3 沟通话术
| 场景 | 话术示例 |
|-----|---------|
| 初次介绍 | "我们是科技管理集团，专注跨境电商AI选品服务，帮助卖家用数据驱动选品决策。" |
| 介绍产品 | "Auto选品是我们的核心产品，通过AI算法分析市场数据，帮您发现蓝海机会。" |
| 强调优势 | "我们的优势是更精准的算法和更懂卖家的产品体验，真正帮您提升选品成功率。" |
| 引导行动 | "欢迎免费试用，体验AI选品的便捷和精准。" |

### 7.4 禁止用语
| 类型 | 示例 |
|-----|-----|
| 绝对化用语 | 最好、第一、顶级 |
| 虚假承诺 | 100%有效、保证赚钱 |
| 贬低竞品 | XX品牌不如我们 |
| 模糊用语 | 可能、也许、大概 |

## 八、品牌管理

|-----|---------|
