## 一、Logo主图标生成

### Prompt 1：扬帆小舟Logo

```
Design a modern minimalist logo for "AutoSelect" - an AI-powered TikTok cross-border e-commerce product selection tool for Chinese sellers entering Southeast Asian markets.

Key elements:
- A stylized geometric boat/ship sailing on wave patterns
- The boat's sail forms an "A" shape (representing "Auto")
- Subtle wave lines beneath representing the ocean
- Enclosed in a partial incomplete circle suggesting global reach

Color palette:
- Primary: Deep ocean blue (#2563EB)
- Accent: Sunrise orange (#F97316)
- White for highlights

Style:
- Minimalist, clean, professional but warm
- Tech-forward with humanistic touch
- Modern flat design with subtle gradients
- Vector-style, clean lines
- Transparent background

Text: "auto select" in clean modern sans-serif font below the icon

Output: PNG with transparent background, square format 1024x1024, suitable for both light and dark backgrounds
```
