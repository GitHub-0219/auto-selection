# Logo设计方案：Auto选品

## 一、设计理念

### 核心理念

**"小舟出海，Auto护航"**

Auto选品的Logo承载着双重含义：
1. **创始人林小舟**的个人IP隐喻——小舟是创始人，也是每一个普通追梦者的化身
2. **Auto（自动化）**的技术承诺——AI驱动，让跨境变得简单

### 设计关键词

| 关键词 | 含义 |
|--------|------|
| 小舟 | 普通人的象征、轻盈感、航行出发 |
| 波浪 | 海洋/跨境、市场波动、动态感 |
| 箭头/导航 | 方向感、指引、AI智能选品 |
| 圆形/闭环 | 完美、闭环服务、全球化 |
| 科技蓝 | 专业、信任、技术感 |

### 设计风格

- **风格定位**：科技感 + 人文温度
- **图形语言**：简洁几何 + 有机曲线结合
- **视觉调性**：专业但不冰冷，亲切但不幼稚

## 三、Logo构图描述

### 方案A：扬帆小舟（主推方案）

**图形元素**：
- 一艘简约几何化的小船，船身由弧线构成，象征"小舟"
- 船底有一条弧形波浪线，代表海洋/跨境
- 船帆位置是一个"A"字形结构，代表Auto首字母
- 整体被一个不完整的圆形包裹，象征全球化闭环

**文字排版**：
- 英文名："AutoSelect" 或 "auto select"
- 中文名："Auto选品"（可选项）

**视觉示意**：

```
        /\
       /  \
      / A  \      ← A字形船帆
     /______\
    /        \
   |  🚢    |    ← 简化小舟
   |________|
 ≈≈≈≈≈≈≈≈≈≈≈≈    ← 波浪线

   [auto select]
```

**设计特点**：
- 小舟呼应创始人IP，辨识度高
- "A"形结构强化Auto品牌
- 圆形包裹暗示全球化视野
- 整体简洁，适合各种尺寸

---

### 方案B：导航箭头

**图形元素**：
- 一个抽象的箭头图标，指向右上方向（代表出海/上升）
- 箭头内部融入"A"字母
- 箭头周围有环绕的虚线圆环

**视觉示意**：

```
      ↗
     ↗ ↗
    ↗   ↗
   ↗  A  ↗
    ↗   ↗
     ↗↗↗
    ○○○○○
```

**设计特点**：
- 方向感强，传达"指引"价值
- 适合工具属性强的产品
- 简洁现代，科技感强

---

### 方案C：智能选品框

**图形元素**：
- 一个带有对勾/星号的方形选框
- 框内融入"A"字母
- 四个角有定位点，暗示"精准定位"

**视觉示意**：

```
    ┌────────┐
    │ • A  ★ │    ← 选品框+精准定位点
    │        │
    │   ✓   │    ← 对勾代表"选对"
    └────────┘
      ·    ·
```

**设计特点**：
- 工具属性明确
- "选品"概念直观
- 适合强调"选品"核心功能

## 四、不同场景应用规范

### 4.1 浅色背景应用

```
背景色：#FFFFFF 或 #F8FAFC
Logo颜色：主色 #2563EB
适用场景：官网首页、白皮书、App Store
```

**注意事项**：
- 确保背景与Logo对比度 ≥ 4.5:1
- 最小使用尺寸：高度 ≥ 32px

---

### 4.2 深色背景应用

```
背景色：#0F172A 或 #1E293B
Logo颜色：#FFFFFF 或 #3B82F6
辅助点缀：#F97316
适用场景：深色官网、Dashboard侧边栏
```

**注意事项**：
- 可选择纯白版本或蓝色渐变版本
- 保持足够辨识度

---

### 4.3 Favicon / App Icon

**规格要求**：
- iOS：180×180px（App Icon）
- Android：192×192px / 512×512px
- Web Favicon：32×32px / 16×16px

**简化设计**：
- 仅使用"A"字形图标
- 或简化小船图形
- 背景使用渐变蓝或纯蓝

**示例结构**：

```
┌─────────────────────┐
│                     │
│    渐变蓝色背景      │
│    ┌─────────┐      │
│    │   A     │      │  ← 白色字母
│    │  ↘↗    │      │  ← 小箭头装饰
│    └─────────┘      │
│                     │
└─────────────────────┘
```

---

### 4.4 社交媒体头像

**平台适配**：

| 平台 | 建议尺寸 | 说明 |
|------|----------|------|
| 微信公众号 | 200×200px | 方形，使用Logo+中文名 |
| 小红书 | 建议圆形裁剪 | 保留核心"A"图标 |
| TikTok | 建议正方形 | 可添加波浪元素 |
| 抖音 | 200×200px | 与TikTok保持一致 |
| 微博 | 200×200px | 标准方形头像 |
| LinkedIn | 品牌页适配 | 保留完整Logo |

---

### 4.5 印刷物料应用

**名片**：
- 颜色：单色版本（纯蓝 #2563EB）
- 尺寸：90×54mm（标准名片）
- 排版：Logo居左/居中，品牌名在下

**宣传册/PPT封面**：
- 颜色：全彩版本
- 可添加品牌Slogan："让出海，像呼吸一样简单"
- 视觉空间充足，可展示完整图形

**周边物料（T恤/马克杯）**：
- 推荐单色丝印版本
- 颜色：白色Logo + 深蓝底 或 蓝色Logo + 白底

## 五、AI生图Prompt

### 5.1 主Logo生成

```
Design a modern logo for "AutoSelect" - an AI-powered TikTok cross-border e-commerce product selection tool targeting Chinese sellers entering Southeast Asian markets.

Style Requirements:
- Minimalist, clean, professional but warm
- Tech-forward with humanistic touch
- Color palette: deep ocean blue (#2563EB) as primary, sunrise orange (#F97316) as accent
- Geometric shapes combined with organic curves

Logo Concept Description:
A stylized small boat (representing "small boat" / 普通人的跨境梦想) sailing on wave patterns. The boat's sail forms an "A" shape (for Auto). Enclosed in a partial circle suggesting global reach. The wave underneath represents the ocean/cross-border journey.

Text: "auto select" in clean sans-serif font below the icon

Technical Requirements:
- Transparent background
- Vector-style, clean lines
- High resolution (2048x2048)
- Modern flat design with subtle gradients
- Professional enough for business use

Output: PNG with transparent background, minimal shadow, suitable for both light and dark backgrounds
```

### 5.2 品牌吉祥物生成

```
Design a cute mascot character for "Auto选品" (AutoSelect) - an AI-powered TikTok e-commerce tool.

Character Concept:
- A friendly cartoon boat character with big expressive eyes
- Wearing a captain's hat with "A" logo on it
- Small size, approachable, lovable like a companion
- Holding a magnifying glass or compass (representing "smart selection")
- Ocean blue color scheme with orange accents
- Expression: determined and helpful

Style:
- Cute kawaii style
- Clean vector illustration
- Simple colors: blue body, orange details, white highlights
- Transparent background
- Friendly, approachable personality

Usage: Brand mascot for social media, tutorials, onboarding screens

Output: PNG with transparent background, character centered, full body visible
```

### 5.3 App图标生成

```
Create an app icon for "Auto选品" (AutoSelect) - a TikTok cross-border e-commerce product research tool.

Design Elements:
- Simplified "A" letter as the main element
- Small wave or boat element subtly integrated
- Smart/tech feel with compass or target overlay

Color Scheme:
- Gradient from deep blue (#2563EB) to cyan (#06B6D4)
- Orange accent (#F97316) for important elements
- White icon on colored background

Style:
- Modern app icon style
- Rounded corners (like iOS/Android standards)
- Gradient background with subtle depth
- Clean, minimal, instantly recognizable at small sizes
- Slight 3D effect or depth for modern feel

Technical:

- Optimized for app store display
- Clean edges, no jaggies
- Includes shadow/depth for polish

Output: PNG, square aspect ratio, high resolution for app store submission
```

## 六、字体规范

### 英文字体

**主字体**：Inter / Noto Sans SC（Google Fonts）
- 用途：正文、界面、文档
- 特点：现代、清晰、多语言支持

**Logo字体**：定制手写体或修改版Montserrat
- 用途：品牌Logo
- 特点：独特、有温度

### 中文字体

**主字体**：思源黑体（Source Han Sans）/ 阿里巴巴普惠体
- 用途：官网、App界面、文档
- 特点：免费商用、清晰易读

**标题字体**：思源宋体
- 用途：品牌故事、重大标题
- 特点：文化感、厚重感

### 字体使用规范

| 场景 | 英文 | 中文 | 字重 |
|------|------|------|------|
| Logo | 定制衬线体 | 思源黑体 Bold | Bold |
| 标题 H1 | Inter Bold | 思源黑体 Bold | 700 |
| 标题 H2 | Inter Semibold | 思源黑体 Bold | 600 |
| 正文 | Inter Regular | 思源黑体 Regular | 400 |
| 辅助说明 | Inter Light | 思源黑体 Light | 300 |

## 七、应用规范汇总

### 品牌安全空间

```
┌─────────────────────────────┐
│                             │
│   ≥ 1xLogo Height           │
│                             │
│  ┌───────┐                  │
│  │       │ ← Logo           │
│  └───────┘                  │
│                             │
│   ≥ 1xLogo Height           │
│                             │
└─────────────────────────────┘

最小安全空间 = Logo高度 × 1
```

### 禁止事项

- ❌ 禁止拉伸或扭曲Logo
- ❌ 禁止改变Logo颜色（除官方允许的深色/浅色版本）
- ❌ 禁止在Logo周围添加未经批准的装饰元素
- ❌ 禁止将Logo放置在对比度不足的背景上
- ❌ 禁止使用过小的尺寸（最小高度32px）

### 版本说明

| 版本 | 使用场景 | 文件格式 |
|------|----------|----------|
| 完整版 | 官网、宣传册 | SVG, PNG, AI |
| 简化版 | App Icon, Favicon | SVG, PNG |
| 单色版 | 单色印刷、名片 | SVG, PNG |
| 深色版 | 深色背景 | SVG, PNG |
