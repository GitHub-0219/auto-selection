# 财务管理制度

## 一、预算管理

### 1.1 预算编制原则
- **总量控制**：根据公司战略目标和年度经营计划确定预算总额
- **零基预算**：每项支出需重新评估，非历史数据的简单延续
- **动态调整**：每季度根据实际业务情况进行预算调整
- **权责对等**：预算责任人与支出审批人保持一致

### 1.2 预算分类
| 类别 | 包含内容 | 审批权限 |
|------|---------|---------|
| 运营预算 | 人力成本、办公费用、营销费用、研发费用 | 部门负责人 |
| 资本预算 | 设备采购、软件采购、服务器成本 | CEO |
| 应急预算 | 未预见性支出 | CEO+财务负责人 |

### 1.3 预算编制流程
1. **预算启动**（每年11月）：
   - 财务部门发布年度预算编制通知
   - 各部门提交预算需求

2. **预算审核**（每年12月前两周）：
   - 财务部门初审
   - CEO办公会审议
   - 确定最终预算

3. **预算下达**（每年12月最后一周）：
   - 正式下发年度预算

4. **执行监控**（每月）：
   - 月度预算执行分析
   - 异常预警

5. **季度回顾**（每季度末）：
   - 预算与实际对比
   - 必要时调整预算

### 1.4 预算控制机制
- **限额控制**：超出预算20%以内需部门负责人审批，超出20%需CEO审批
- **月度预警**：预算执行率>80%时触发预警
- **红绿灯机制**：
  - 🟢 绿色：执行率≤85%
  - 🟡 黄色：85% < 执行率 ≤100%
  - 🔴 红色：执行率 >100%

## 二、成本控制

### 2.1 成本控制原则（参考丰田精益管理）
- **消除浪费**：识别并消除不增值的活动
- **持续改进**：通过PDCA循环持续优化
- **全员参与**：每位员工都是成本控制的责任人
- **数据驱动**：基于数据进行决策，而非经验

### 2.2 成本分类管控
| 成本类型 | 控制策略 | 关键指标 |
|---------|---------|---------|
| 人力成本 | 人效导向，控制人员增长 | 人均产出、人均利润 |
| 技术成本 | 云资源优化、复用优先 | 云成本/营收比、服务器利用率 |
| 营销成本 | ROI导向，精准投放 | CAC、LTV/CAC |
| 办公成本 | 共享经济、按需采购 | 人均办公成本 |
| 研发成本 | 核心技术优先，避免重复造轮子 | 研发产出比 |

### 2.3 成本优化措施
1. **技术成本优化**：
   - 云资源自动伸缩
   - 定期清理僵尸实例
   - 使用预付费资源包
   - 技术选型成本评估

2. **人力成本优化**：
   - 核心岗位自建，辅助岗位外包
   - 提升人效而非简单增加人数
   - 合理使用实习生和兼职

3. **营销成本优化**：
   - 建立ROI跟踪体系
   - 聚焦高转化渠道
   - 内容营销优于付费广告

### 2.4 成本节约激励
- 设立"成本节约奖"，奖励成本优化建议
- 成本节约金额的10%用于团队奖励
- 每月评选"降本增效之星"

## 三、收入管理

### 3.1 收入确认原则
- **权责发生制**：按合同约定确认收入
- **完工百分比法**：长期服务合同按进度确认
- **风险转移**：以产品/服务交付给客户为确认时点

### 3.2 收入分类
| 收入类型 | 确认标准 | 跟踪方式 |
|---------|---------|---------|
| 订阅收入 | 按月/季/年确认 | 财务系统自动 |
| 项目收入 | 按里程碑确认 | 项目管理系统 |
| 增值服务 | 按交付确认 | 客工单系统 |
| 代理/分销 | 按结算周期确认 | 渠道管理系统 |

### 3.3 应收账款管理
1. **账期管理**：
   - SaaS订阅：预付费
   - 项目服务：30-60天账期
   - 大客户：可协商最长90天

2. **催收机制**：
   - 到期前3天提醒
   - 逾期7天发送正式通知
   - 逾期15天电话沟通
   - 逾期30天暂停服务
   - 逾期60天法律介入

3. **坏账准备**：
   - 逾期30天：计提20%
   - 逾期60天：计提50%
   - 逾期90天：计提100%

## 四、审批流程（扁平化设计）

### 4.1 审批权限矩阵
| 金额范围 | 部门负责人 | 财务负责人 | CEO | 董事会 |
|---------|-----------|-----------|-----|-------|
| ≤ 5,000元 | ✅ | - | - | - |
| 5,001-20,000元 | ✅ | ✅ | - | - |
| 20,001-100,000元 | ✅ | ✅ | ✅ | - |
| 100,001-500,000元 | ✅ | ✅ | ✅ | 备案 |
| >500,000元 | ✅ | ✅ | ✅ | ✅ |

### 4.2 审批流程
1. **申请**：填写《支出申请模板》
2. **初审**：部门负责人审核必要性和合理性
3. **财务审核**：财务负责人审核合规性、预算内
4. **最终审批**：根据金额权限审批
5. **付款**：财务部门执行付款

### 4.3 审批时效
- 常规审批：2个工作日内完成
- 紧急事项：4小时内完成
- 重大事项：7个工作日内完成

### 4.4 审批简化原则
- 5,000元以下：简化流程，只需部门负责人审批
- 预算内常规支出：授权审批，每月集中备案
- 重复性支出（如服务器续费）：年度授权，自动续费

## 五、资金预警机制

### 5.1 监控指标
| 指标 | 预警阈值 | 危险阈值 | 责任人 |
|-----|---------|---------|-------|
| 现金储备月数 | < 6个月 | < 3个月 | CEO+财务负责人 |
| 月度现金流 | 连续2个月负 | 连续3个月负 | 财务负责人 |
| 应收账款周转天数 | > 60天 | > 90天 | 销售负责人 |
| 预算执行率 | > 95% | > 100% | 部门负责人 |

### 5.2 预警级别
- **黄色预警**：触发一个指标，相关部门需提交改进计划
- **橙色预警**：触发两个指标或一个危险指标，CEO需亲自过问
- **红色预警**：触发三个以上危险指标，启动紧急应对机制

### 5.3 应对措施
1. **现金储备不足**：
   - 暂停非必要支出
   - 加大应收账款催收力度
   - 启动融资准备

2. **现金流紧张**：
   - 推迟可延后支出
   - 寻求短期融资
   - 促销加速现金回笼

3. **应收账款逾期**：
   - 销售部门专人跟进
   - 调整客户信用政策
   - 必要时法律手段

## 六、报销管理

### 6.1 报销原则
- **业务必要**：必须与公司业务相关
- **真实合规**：票据真实，符合税务要求
- **及时性**：费用发生后30天内报销
- **预算内**：不得超出预算

### 6.2 报销标准
| 费用类型 | 标准 | 备注 |
|---------|-----|-----|
| 交通费 | 实报实销 | 市内交通限滴滴专车 |
| 差旅住宿 | 一线城市≤500元/晚 | 超出部分自理 |
| 餐饮费 | ≤100元/人/次 | 需注明用餐事由 |
| 招待费 | ≤200元/人/次 | 需提前申请 |
| 办公用品 | ≤200元/单 | 超出需采购 |

### 6.3 报销流程
1. 提交报销申请（附票据）
2. 部门负责人审批
3. 财务审核
4. 付款（3个工作日内）

## 七、财务报告

### 7.1 报告周期
- **月度报告**：每月5日前完成
- **季度报告**：每季度首月15日前完成
- **年度报告**：每年2月前完成

### 7.2 月度报告内容
1. 财务状况（资产负债表、利润表、现金流量表）
2. 预算执行分析
3. 关键财务指标（营收、利润、现金流）
4. 异常说明与改进建议

### 7.3 财务分析会
- **频率**：每月10日召开
- **参会人**：CEO、各部门负责人、财务负责人
- **议程**：
  1. 财务数据汇报（15分钟）
  2. 预算执行情况（10分钟）
  3. 问题分析与讨论（20分钟）
  4. 改进措施确认（15分钟）

## 八、合规管理

### 8.1 税务合规
- 按时申报纳税
- 规范发票管理
- 合理税务筹划
- 保留完整凭证

### 8.2 内控管理
- 职责分离（申请、审批、付款分离）
- 定期审计（每年一次外部审计）
- 关键岗位轮岗（财务关键岗位2年一轮换）

### 8.3 反舞弊
- 建立举报机制
- 零容忍原则
- 发现即开除+追究法律责任
