# 内容创作规范

> 科技管理集团 - 精品内容生产标准

---

## 一、创作理念

> **休息时不闲置，转向内容创作**
> **细细打磨，逐帧优化，追求极致**

### 核心原则
1. **精品优先**：宁缺毋滥，不做粗制滥造
2. **细节打磨**：每一帧、每一句都要推敲
3. **用户价值**：内容必须真正帮助用户
4. **持续迭代**：根据反馈不断优化

## 三、创作流程

### 3.1 视频创作流程

```
选题策划 → 脚本撰写 → 素材准备 → 录制/制作 → 粗剪 → 精剪 → 审核修改 → 发布
   ↓          ↓          ↓          ↓         ↓       ↓        ↓        ↓
 用户需求   逐字打磨   高质量素材  专业设备   整体节奏 逐帧优化  多人审核  多渠道
```

### 3.2 逐帧优化标准

| 环节 | 优化要点 |
|------|----------|
| **画面** | 清晰度、色彩、构图、转场 |
| **声音** | 音质、音量、背景音乐、音效 |
| **字幕** | 准确性、可读性、美观度 |
| **节奏** | 剪辑节奏、信息密度、观看体验 |
| **内容** | 逻辑性、实用性、吸引力 |

### 3.3 质量检查清单

**发布前必须检查**：
- [ ] 内容准确性（数据、信息无误）
- [ ] 画面质量（1080P以上）
- [ ] 声音质量（无杂音、音量适中）
- [ ] 字幕完整（无遗漏、无错别字）
- [ ] 版权合规（素材、音乐、字体）
- [ ] 品牌统一（Logo、配色、话术）

## 六、内容审核

### 6.1 审核标准

| 维度 | 标准 | 权重 |
|------|------|------|
| **内容质量** | 准确、有价值、逻辑清晰 | 40% |
| **制作质量** | 画面、声音、字幕、节奏 | 30% |
| **用户价值** | 解决问题、有启发、可执行 | 20% |
| **品牌一致** | 话术、视觉、调性统一 | 10% |

### 6.2 审核流程

```
创作者自审 → 同事互审 → 产品经理审核 → 发布
    ↓           ↓            ↓          ↓
 质量检查    内容建议     最终确认    多渠道
```
