# 人力资源制度

## 一、招聘管理

### 1.1 招聘原则
- **岗位匹配**：以岗定人，确保能力与岗位要求匹配
- **德才兼备**：品德优先，能力其次
- **内部优先**：同等条件下，优先考虑内部员工
- **效率优先**：快速决策，不因流程影响人才获取

### 1.2 招聘流程
```
需求提出 → 岗位JD发布 → 简历筛选 → 初试（部门负责人） → 复试（交叉面试） → 背景调查 → 录用审批 → Offer发放
```

### 1.3 各阶段要求
| 阶段 | 时效要求 | 责任人 | 产出物 |
|-----|---------|-------|-------|
| 需求提出 | 用人需求产生后3天内 | 部门负责人 | 招聘需求表 |
| JD发布 | 需求确认后1天内 | HR | 岗位说明书 |
| 简历筛选 | 收到简历后2天内 | HR+部门负责人 | 筛选名单 |
| 初试 | 简历通过后3天内 | 部门负责人 | 面试评估表 |
| 复试 | 初试通过后3天内 | 部门负责人+CEO/或其他部门 | 综合评估表 |
| 背景调查 | 复试通过后3天内 | HR | 背景调查报告 |
| Offer发放 | 背景调查通过后2天内 | HR | Offer函 |

### 1.4 面试评估标准
| 维度 | 权重 | 说明 |
|-----|-----|-----|
| 岗位技能 | 40% | 专业能力与岗位要求匹配度 |
| 潜力 | 25% | 学习能力、成长空间 |
| 文化匹配 | 25% | 与公司价值观的契合度 |
| 团队协作 | 10% | 沟通能力、协作意识 |

### 1.5 试用期管理
- **试用期时长**：3个月（劳动合同3年以上6个月）
- **试用期考核**：
  - 第1个月：周回顾
  - 第2个月：月评估
  - 第3个月：转正评审
- **转正标准**：
  - 完成任务达标
  - 价值观认同
  - 直属负责人认可
- **转正流程**：员工提交转正申请 → 直属负责人评价 → HR审核 → 转正生效

### 1.6 人才画像示例
| 岗位 | 硬技能 | 软技能 | 加分项 |
|-----|-------|-------|-------|
| 产品经理 | 需求分析、产品设计、数据分析 | 沟通表达、逻辑思维 | 有AI产品经验、跨境电商背景 |
| 研发工程师 | 编程能力、架构设计、代码质量 | 自驱力、学习能力、团队协作 | 开源贡献、技术博客 |
| 运营专员 | 数据分析、营销策划、内容运营 | 执行力、细致耐心 | 有SaaS产品运营经验 |
| 设计师 | UI/UX设计、交互能力、视觉审美 | 沟通表达、抗压能力 | 有B端产品经验 |

## 二、考核机制（OKR + KPI 融合）

### 2.1 考核理念（参考字节跳动OKR）
- **目标导向**：聚焦关键结果，而非过程
- **透明公开**：全公司OKR公开可见
- **上下对齐**：个人OKR承接团队/公司OKR
- **挑战性**：目标设定需有一定挑战性，完成70%即为优秀
- **去权威**：OKR不是考核工具，是协作工具

### 2.2 OKR周期
| 周期 | 制定时间 | 评审时间 | 回顾时间 |
|-----|---------|---------|---------|
| 年度OKR | 每年12月 | 每年1月第一周 | 每月 |
| 季度OKR | 每季度末月 | 下季度首周 | 每月 |

### 2.3 OKR制定流程
1. **公司级OKR制定**（CEO牵头）：
   - 分析战略目标
   - 确定3-5个核心O（Objective）
   - 每个O设定2-4个KR（Key Result）

2. **部门级OKR制定**：
   - 承接公司级OKR
   - 结合部门职责
   - 部门内充分讨论

3. **个人OKR制定**：
   - 承接部门OKR
   - 结合岗位职责
   - 与直属上级沟通确认

### 2.4 OKR模板
```
Objective（目标）: 
- 描述一个鼓舞人心的目标
- 使用"动词+名词"的格式
- 具有挑战性但可达成

Key Results（关键结果）:
KR1: [衡量指标]，从[当前值]到[目标值]
KR2: [衡量指标]，从[当前值]到[目标值]
KR3: [衡量指标]，从[当前值]到[目标值]

 Initiatives（关键行动，可选）:
- 为达成KR需要做的具体行动
```

### 2.5 OKR评分标准
| 得分 | 含义 | 说明 |
|-----|-----|-----|
| 1.0 | 卓越 | 完全达成，甚至超越预期 |
| 0.7 | 优秀 | 很好地完成了目标 |
| 0.5 | 合格 | 完成了基础目标 |
| 0.3 | 需改进 | 未达成基本目标 |
| 0 | 未开始 | 未采取任何行动 |

**评分原则**：
- 0.6-0.7分是"努力可以够到"的目标
- 平均分0.6-0.7是正常状态
- 连续低于0.5需要复盘原因

### 2.6 KPI指标（辅助考核）
| 岗位类别 | 核心KPI | 说明 |
|---------|--------|-----|
| 研发 | 代码质量、按时交付、技术债务 | 研发产出相关指标 |
| 产品 | 产品完成度、用户满意度、数据指标 | 产品价值相关指标 |
| 运营 | 用户增长、转化率、留存率 | 增长相关指标 |
| 销售 | 营收、回款、客户数 | 收入相关指标 |

### 2.7 考核周期与形式
| 考核类型 | 周期 | 形式 | 结果应用 |
|---------|-----|-----|---------|
| OKR回顾 | 月度 | 自评+上级辅导 | 发现问题，及时调整 |
| OKR评审 | 季度 | 公开述职 | 对齐方向，持续改进 |
| 综合评估 | 半年/年度 | 360度评估 | 晋升、调薪、培训依据 |

## 三、激励制度

### 3.1 薪酬理念（参考华为"奋斗者为本"）
- **以贡献定回报**：薪酬与价值创造直接挂钩
- **拉开差距**：核心岗位薪酬领先市场
- **即时激励**：及时认可，及时奖励
- **全面激励**：物质+精神双激励

### 3.2 薪酬结构
| 组成部分 | 说明 | 比例 |
|---------|-----|-----|
| 基本工资 | 岗位价值体现 | 固定 |
| 绩效奖金 | 与个人/公司业绩挂钩 | 浮动 |
| 年终奖金 | 与公司全年业绩挂钩 | 年度 |
| 股权激励 | 核心员工，长期绑定 | 长期 |

### 3.3 绩效奖金
| 绩效等级 | 占比 | 奖金系数 | 说明 |
|---------|-----|---------|-----|
| A（卓越） | 10% | 2.0x | 超预期贡献 |
| B（优秀） | 25% | 1.5x | 超出预期 |
| C（达标） | 50% | 1.0x | 完成预期 |
| D（待改进） | 10% | 0.5x | 未完全达标 |
| E（不合格） | 5% | 0 | 未达标 |

### 3.4 即时激励
| 激励类型 | 金额/形式 | 触发条件 | 审批人 |
|---------|----------|---------|-------|
| 表扬信 | 公开表扬 | 出色完成工作 | 直属上级 |
| 小额奖励 | 100-500元 | 创新贡献 | 部门负责人 |
| 最佳实践奖 | 500-2000元 | 方法创新推广 | CEO |
| 项目成功奖 | 1000-5000元 | 重大项目完成 | CEO |

### 3.5 晋升机制
| 晋升通道 | 职级序列 | 晋升周期 |
|---------|---------|---------|
| 管理通道 | 骨干→主管→经理→总监→VP | 1-2年/级 |
| 专业通道 | 初级→中级→高级→专家→资深专家 | 1-2年/级 |

**晋升标准**：
- 绩效要求：连续2年绩效B以上
- 能力要求：达到下一职级能力要求
- 机会要求：有空缺岗位或编制

### 3.6 股权激励
- **授予对象**：核心管理层、关键岗位（VP/总监级别及以上）
- **授予节奏**：4年成熟期，每年25%
- **行权条件**：在职、达成业绩目标
- **退出机制**：离职需按协议回购

## 四、培训发展

### 4.1 培训体系（参考Google 70-20-10法则）
- **70%**：实践中学习（项目历练、轮岗）
- **20%**：向他人学习（导师制、经验分享）
- **10%**：正式培训（外部课程、内部培训）

### 4.2 培训类型
| 类型 | 形式 | 时长 | 频率 |
|-----|-----|-----|-----|
| 新人培训 | 集中培训+导师制 | 1周+3个月带教 | 入职即开始 |
| 专业培训 | 内部分享+外部课程 | 按需 | 每月至少1次 |
| 管理培训 | 案例教学+实战练习 | 系统性课程 | 每季度 |
| 通识培训 | 读书会、分享会 | 1-2小时 | 每周 |

### 4.3 新人培训内容
| 时间 | 内容 | 负责人 |
|-----|-----|-------|
| 第1天 | 公司介绍、团队认识、工具安装 | HR |
| 第2-3天 | 产品/业务学习 | 部门负责人 |
| 第1周 | 业务流程、基础任务上手 | 导师 |
| 第1个月 | 独立承担基础任务 | 导师+部门负责人 |
| 第3个月 | 独立承担完整工作 | 导师 |

### 4.4 导师制
- **导师匹配**：入职1周内完成匹配
- **导师职责**：业务指导、文化传承、职业发展
- **沟通频率**：每周至少1次正式沟通
- **考核激励**：优秀导师给予奖励

### 4.5 学习资源
- **内部**：
  - 技术/业务分享会
  - 知识库文档
  - 经验复盘
- **外部**：
  - 得到/混沌等课程
  - 行业会议
  - 专业书籍（公司报销）

### 4.6 培训费用
| 类型 | 报销标准 | 审批 |
|-----|---------|-----|
| 专业书籍 | 200元/月 | 直属上级 |
| 在线课程 | 500元/季度 | 部门负责人 |
| 外部培训 | 3000元/年 | CEO |
| 行业会议 | 5000元/年 | CEO |

## 五、员工关系

### 5.1 入离职管理
| 环节 | 要求 | 责任人 |
|-----|-----|-------|
| 入职 | 签署合同、保密协议、员工手册确认 | HR |
| 转正 | 试用期满前1周启动 | 部门+HR |
| 调岗 | 双方同意，审批通过 | 部门+HR |
| 离职 | 提前30天申请，工作交接 | 部门+HR |

### 5.2 考勤制度
- **工作时间**：弹性工作制，核心时间10:00-16:00
- **出勤要求**：每月满勤22天
- **请假类型**：事假、病假、年假、婚假、产假等
- **请假流程**：提前申请，审批通过后生效

### 5.3 福利制度
| 福利类型 | 标准 | 说明 |
|---------|-----|-----|
| 五险一金 | 按法律规定 | 全额缴纳 |
| 午餐补贴 | 20元/天 | 工作日 |
| 通讯补贴 | 100-300元/月 | 按岗位 |
| 交通补贴 | 特殊岗位 | 按需 |
| 年度体检 | 1000元/年 | 基础体检 |
| 团建活动 | 200元/人/月 | 部门活动 |
| 节日福利 | 春节、中秋等 | 礼品/红包 |
| 带薪年假 | 5-15天 | 按司龄 |

### 5.4 离职管理
- **主动离职**：提前30天书面申请，工作交接
- **被动离职**：根据情况确定通知期
- **离职面谈**：了解离职原因，收集改进建议
- **离职交接**：完整交接工作、资产、权限
