## 1. 注册前准备工作

### 1.1 确定公司基本信息

| 序号 | 项目 | 建议内容 | 备注 |
|------|------|----------|------|
| 1 | **公司名称** | 科技管理集团有限公司 | 准备3-5个备选名称 |
| 2 | **注册资本** | 500万元 | 可根据实际情况调整 |
| 3 | **经营范围** | 技术服务、软件开发、货物进出口等 | 参考同行并结合业务规划 |

| 5 | **公司类型** | 股份有限公司（发起设立） | 适用初创企业 |
| 6 | **经营期限** | 长期（20年起） | 可约定长期或固定期限 |

### 1.2 确定股权结构

| 序号 | 工作内容 | 输出文件 | 注意事项 |
|------|----------|----------|----------|
| 1 | 确定发起人名单及人数 | 发起人名单 | 1-200人，需有过半数在中国境内有住所 |
| 2 | 制定股权分配方案 | 股权结构表 | 建议创始人持股不低于67%（绝对控制权） |
| 3 | 确定注册资本认缴金额 | 出资确认表 | 各发起人认缴金额及比例 |
| 4 | 起草发起人协议 | 发起人协议 | 明确各方权利义务 |

### 1.3 确定治理结构

| 序号 | 工作内容 | 注意事项 |
|------|----------|----------|
| 1 | 确定董事会成员人数 | 5-19人 |
| 2 | 确定监事会成员人数 | 3人以上，需有职工代表（不低于1/3） |
| 3 | 确定管理层人选 | 总经理、财务负责人等 |
| 4 | 起草公司章程 | 股份有限公司章程 |

### 1.4 确定注册地址

| 项目 | 要求 | 建议 |
|------|------|------|
| **地址性质** | 商用地址（部分园区可用虚拟地址） | 优先选择实际经营地址 |

| **虚拟地址** | 部分地区支持 | 注意后续银行开户可能受限 |

## 2. 公司注册流程

### 2.1 整体流程图

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        股份制公司注册流程                                │
└─────────────────────────────────────────────────────────────────────────┘

第1阶段：注册前准备（预计5-10个工作日）
    │
    ├── 确定公司名称 → 名称核准
    ├── 确定股权结构 → 发起人协议
    ├── 制定公司章程
    ├── 确定注册地址
    └── 准备相关人员证件
    │
    ▼
第2阶段：工商注册（预计5-7个工作日）
    │
    ├── 在线提交预审 → 提交材料
    ├── 窗口递交材料 → 审核
    ├── 领取营业执照
    │
    ▼
第3阶段：后续登记（预计3-5个工作日）
    │
    ├── 刻制印章
    ├── 银行开户
    ├── 税务登记
    └── 社保/公积金开户
    │
    ▼
第4阶段：初始化运营（预计5-10个工作日）
    │
    ├── 开设相关账户
    ├── 建立财务账套
    ├── 股权激励计划备案（如适用）
    └── 其他资质许可
```

### 2.2 详细步骤

#### 步骤1：企业名称核准

| 项目 | 说明 |
|------|------|
| **办理部门** | 市场监督管理局/政务服务网 |
| **办理方式** | 线上（全国一体化政务服务平台）或线下窗口 |
| **核名规则** | 行政区划+字号+行业+组织形式 |
| **有效期** | 名称核准通知书有效期6个月 |
| **注意事项** | 准备3-5个备选名称，避免近似 |

#### 步骤2：提交注册材料

| 项目 | 说明 |
|------|------|
| **提交方式** | 线上提交+线下窗口递交 |
| **审核时间** | 3-5个工作日 |
| **补正通知** | 如材料不全，收到补正通知后补充 |

#### 步骤3：领取营业执照

| 项目 | 说明 |
|------|------|
| **领取方式** | 窗口领取或邮寄 |
| **证照内容** | 正本1份，副本若干 |
| **电子执照** | 同步生成电子营业执照 |

#### 步骤4：刻制印章

| 印章类型 | 数量 | 说明 |
|----------|------|------|
| **公章** | 1枚 | 必刻，公司主体印章 |
| **财务章** | 1枚 | 必刻，财务专用 |
| **法人章** | 1枚 | 必刻，法人代表印鉴 |
| **发票章** | 1枚 | 如需开具发票 |
| **合同章** | 1枚 | 可选，合同专用 |

**办理部门**：公安局指定刻章点

#### 步骤5：银行开户

| 项目 | 说明 |
|------|------|
| **基本户** | 1个，必开，用于日常结算 |
| **一般户** | 多个，可选，用于专项业务 |
| **外汇户** | 1个，跨境电商必需 |
| **办理时间** | 3-7个工作日 |
| **注意事项** | 需携带营业执照、公章、法人身份证等 |

#### 步骤6：税务登记

| 项目 | 说明 |
|------|------|
| **登记时限** | 领取营业执照30日内 |
| **登记内容** | 税种、税率、申报方式 |
| **购票人** | 需实名认证 |
| **注意事项** | 按规定申报纳税，否则面临处罚 |

#### 步骤7：社保/公积金开户

| 项目 | 说明 |
|------|------|
| **开户部门** | 社保局、公积金管理中心 |
| **办理时限** | 录用员工30日内 |
| **办理内容** | 单位账户设立、员工参保 |

## 4. 办理时间预估

### 4.1 各环节时间

| 序号 | 事项 | 办理方式 | 预计时间 | 备注 |
|------|------|----------|----------|------|
| 1 | **企业名称核准** | 线上/线下 | 1-2个工作日 | 可同时准备其他材料 |
| 2 | **工商注册** | 线上提交+窗口 | 3-5个工作日 | 材料齐全情况下 |
| 3 | **领取执照** | 窗口/邮寄 | 1个工作日 | 或当场领取 |
| 4 | **刻制印章** | 指定刻章点 | 1个工作日 | 部分地区可当日 |
| 5 | **银行开户** | 银行网点 | 3-7个工作日 | 视银行而定 |
| 6 | **税务登记** | 线上/线下 | 1-2个工作日 | 领取执照30日内 |
| 7 | **社保开户** | 社保局 | 1-2个工作日 | 录用员工前完成 |
| 8 | **公积金开户** | 公积金中心 | 1-2个工作日 | 录用员工前完成 |

### 4.2 总时间预估

| 场景 | 最短时间 | 正常时间 | 最长时间 |
|------|----------|----------|----------|
| **顺利情况下** | 10个工作日 | 15-20个工作日 | 25-30个工作日 |
| **含特殊情况** | 15个工作日 | 20-30个工作日 | 45-60个工作日 |

### 4.3 时间影响因素

| 因素 | 影响说明 | 应对建议 |
|------|----------|----------|
| 材料准备 | 完整材料加快进度 | 提前准备齐全 |
| 名称核准 | 名称重复需重新核准 | 多备选名称 |
| 地址问题 | 地址异常需核实 | 使用合规地址 |
| 银行审核 | 尽调严格延长时间 | 选择配合度高的银行 |
| 政策性因素 | 政策调整影响 | 关注最新政策 |
