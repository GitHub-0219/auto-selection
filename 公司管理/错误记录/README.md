# 错误记录目录

本目录用于记录所有工作中的错误和失误，以便追踪和改进。

## 记录原则
1. **实事求是**：不隐瞒、不夸大
2. **及时记录**：发现错误立即记录
3. **追踪到底**：记录修复过程和预防措施
4. **定期复盘**：每周汇总分析

## 文件命名规范
- 单次错误：`错误记录_YYYYMMDD_{编号}.md`
- 重复错误：`错误记录_YYYYMMDD_{编号}_重复.md`
