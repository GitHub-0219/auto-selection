## AI智能选品

AI选品是平台的核心功能，帮助您发现高潜力的跨境电商商品。

### 功能入口

登录后点击顶部导航「AI选品」或「用户中心」→「AI智能选品」

### 使用步骤

#### 步骤1：选择目标平台

支持的平台：

| 平台 | 说明 | 覆盖地区 |
|------|------|----------|
| Shopee | 虾皮购物 | 东南亚6国 |
| Lazada | 来赞达 | 东南亚6国 |
| TikTok Shop | TikTok电商 | 全球 |

#### 步骤2：输入商品关键词

输入您想分析的关键词，例如：
- 蓝牙耳机
- 智能手表
- 无线充电器
- 运动相机
- 美妆工具

#### 步骤3：设置分析参数（可选）

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 商品类目 | 限定商品所属类目 | 不限制 |
| 返回数量 | 期望的分析结果数量 | 10个 |

#### 步骤4：开始分析

点击「开始分析」按钮，系统将自动：

1. 🔍 搜索目标平台相关商品
2. 📊 分析销量、评分、评价数量
3. 🤖 AI评估市场竞争度
4. 💡 生成选品建议

## 多语言翻译

多语言翻译功能支持50+语言的自动翻译，适合商品标题、描述的本地化处理。

### 功能入口

点击顶部导航「翻译」或「用户中心」→「多语言翻译」

### 快速翻译

#### 单条翻译

1. 选择源语言（支持自动检测）
2. 选择目标语言
3. 输入要翻译的文本
4. （可选）选择翻译风格
5. 点击「翻译」

**翻译风格说明：**

| 风格 | 适用场景 | 示例 |
|------|----------|------|
| 正式 | 官方文档、商务场景 | "We provide..." |
| 口语 | 社交媒体、日常交流 | "We've got..." |
| 营销 | 商品推广、广告文案 | "Amazing! Unbeatable..." |

#### 批量翻译

1. 点击「批量翻译」标签
2. 逐行输入要翻译的文本（最多20条）
3. 选择源语言和目标语言
4. 点击「批量翻译」

### 支持的语言

| 目标市场 | 语言 |
|----------|------|
| 东南亚 | 泰语、越南语、印尼语、马来语、菲律宾语 |
| 欧美 | 英语、法语、德语、西班牙语、意大利语、葡萄牙语 |
| 东亚 | 日语、韩语 |
| 中东 | 阿拉伯语 |

### 使用示例

**场景：翻译商品标题到英语（营销风格）**

输入：
```
高品质无线蓝牙耳机，支持主动降噪功能
```

输出：
```
✨ High-Quality Wireless Bluetooth Earbuds with Active Noise Cancellation ✨
```

### 翻译技巧

1. **保持简洁**：短句翻译更准确
2. **避免歧义**：专业术语可保留原文
3. **检查校对**：重要内容建议人工校对
4. **批量优化**：批量翻译时可先生成再调整

### 积分消耗

| 操作类型 | 消耗积分 |
|----------|----------|
| 单条翻译（简短） | 5-10积分 |
| 单条翻译（长文本） | 15-30积分 |
| 批量翻译 | 按条数累计 |

智能定价帮助您制定最优的商品售价策略。

### 功能入口

点击顶部导航「定价」或「用户中心」→「智能定价」

### 定价分析

1. **商品名称**：输入待定价商品名称
2. **成本价格**：输入您的进货成本

4. **竞品价格**：（可选）输入已知竞品价格


### 定价策略说明

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 竞争性定价 | 低于市场均价10%左右 | 新品推广、市场开拓 |
| 溢价定价 | 高于市场均价 | 差异化产品、品质保证 |
| 渗透定价 | 低于成本价或接近成本 | 快速获取市场份额 |
| 动态定价 | 根据实时数据调整 | 竞品监控、促销活动 |

### 盈亏平衡计算

**公式**：
```
盈亏平衡价 = 成本价 ÷ (1 - 平台费率 - 运费占比)
```

**示例**：
- 成本：$15.00
- 平台费率：14%（佣金10%+交易费1%+支付费3%）
- 运费：$3.00

```
盈亏平衡价 = $15.00 ÷ (1 - 14% - $3.00)
           = $15.00 ÷ 0.86
           = $17.44
```

### 定价建议

1. **参考竞品**：分析同类商品的定价区间
2. **计算成本**：包含商品成本、平台费用、运费
3. **预留空间**：为促销活动预留降价空间
4. **定期调整**：根据市场变化动态调整价格
