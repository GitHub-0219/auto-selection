## 方式一：一键部署

适用于 Ubuntu/Debian 服务器，快速完成全部部署。

### 步骤

```bash
# 1. SSH 连接服务器
ssh user@139.226.175.192

# 2. 下载部署脚本
cd /opt
sudo git clone https://github.com/GitHub-0219/auto-selection.git
cd auto-selection/project

# 3. 执行部署脚本
chmod +x deploy.sh
sudo ./deploy.sh

# 4. 按照提示完成配置
```

部署脚本会自动完成：
- 安装 Node.js 和 npm
- 安装 PM2 进程管理器
- 安装 Nginx
- 配置防火墙
- 克隆代码仓库
- 安装依赖
- 构建项目
- 配置 Nginx
- 启动服务

## 方式二：手动部署

### 步骤 1：安装基础软件

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# 验证安装
node -v  # v18.x.x
npm -v   # 9.x.x

# 安装 PM2
sudo npm install -g pm2

# 安装 Nginx
sudo apt install -y nginx

# 启动 Nginx
sudo systemctl start nginx
sudo systemctl enable nginx

# 安装 PostgreSQL (可选，如使用数据库)
sudo apt install -y postgresql postgresql-contrib
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

### 步骤 2：创建项目目录

```bash
# 创建项目目录
sudo mkdir -p /var/www/auto-selection
sudo chown -R $USER:$USER /var/www/auto-selection

# 进入目录
cd /var/www/auto-selection

# 克隆代码
sudo git clone https://github.com/GitHub-0219/auto-selection.git .
sudo chown -R $USER:$USER /var/www/auto-selection
```

### 步骤 3：部署后端

```bash
# 进入后端目录
cd backend

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
nano .env  # 编辑配置

# 构建项目
npm run build

# 使用 PM2 启动
pm2 start dist/main.js --name "auto-selection-backend"

# 设置开机自启
pm2 startup
pm2 save
```

### 步骤 4：部署前端

```bash
# 进入前端目录
cd ../frontend

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
nano .env  # 编辑配置

# 构建项目
npm run build
```

### 步骤 5：配置 Nginx

```bash
# 创建 Nginx 配置文件
sudo nano /etc/nginx/sites-available/auto-selection
```

添加以下内容：

```nginx
server {
    listen 80;
    server_name _;
    
    # 前端静态文件
    location / {
        root /var/www/auto-selection/frontend;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # 后端 API 代理
    location /api {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：

```bash
# 启用站点
sudo ln -sf /etc/nginx/sites-available/auto-selection /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重载 Nginx
sudo systemctl reload nginx
```

### 步骤 6：配置防火墙

```bash
# 允许 SSH (重要!)
sudo ufw allow 22/tcp

# 允许 HTTP/HTTPS
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp

# 启用防火墙
sudo ufw enable
```

## 方式三：Docker 部署

### 前提条件

安装 Docker 和 Docker Compose：

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 安装 Docker Compose
sudo apt install -y docker-compose

# 验证安装
docker --version
docker-compose --version
```

### 部署步骤

```bash
# 进入项目目录
cd project

# 使用 Docker Compose 启动
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### Docker Compose 配置说明

```yaml
version: '3.8'

services:
  backend:
    build: ./backend
    ports:
      - "3001:3001"
    environment:
      - NODE_ENV=production
      - DATABASE_URL=postgresql://user:password@db:5432/auto_selection
    depends_on:
      - db
    restart: unless-stopped

  frontend:
    build: ./frontend
    ports:
      - "3000:3000"
    depends_on:
      - backend
    restart: unless-stopped

  db:
    image: postgres:14-alpine
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
      - POSTGRES_DB=auto_selection
    volumes:
      - postgres_data:/var/lib/postgresql/data
    restart: unless-stopped

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
    volumes:
      - ./nginx.conf:/etc/nginx/conf.d/default.conf
      - ./frontend/out:/usr/share/nginx/html
    depends_on:
      - frontend
      - backend
    restart: unless-stopped

volumes:
  postgres_data:
```

## 服务管理

### PM2 命令

```bash
# 查看服务状态
pm2 status

# 查看日志
pm2 logs auto-selection-backend

# 重启服务
pm2 restart auto-selection-backend

# 停止服务
pm2 stop auto-selection-backend

# 删除服务
pm2 delete auto-selection-backend

# 保存当前状态
pm2 save

# 查看实时日志
pm2 logs --lines 100
```

### Systemd 服务 (可选)

创建服务文件 `/etc/systemd/system/auto-selection.service`：

```ini
[Unit]
Description=Auto Selection Backend
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/var/www/auto-selection/backend
ExecStart=/usr/bin/node dist/main.js
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启用服务：

```bash
sudo systemctl daemon-reload
sudo systemctl enable auto-selection
sudo systemctl start auto-selection
```

## 常见问题

### 1. 端口被占用

```bash
# 查找占用端口的进程
sudo lsof -i :3001
sudo lsof -i :80

# 终止进程
sudo kill -9 <PID>
```

### 2. 权限问题

```bash
# 修复目录权限
sudo chown -R www-data:www-data /var/www/auto-selection
sudo chmod -R 755 /var/www/auto-selection
```

### 3. 内存不足

```bash
# 增加 swap
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```
