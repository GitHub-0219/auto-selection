## 二、竞品深度分析

### 2.1 竞品对比矩阵

| 维度 | GoBox | 卖家穿海 | 1688遨虾 | Jungle Scout |
|------|-------|---------|----------|-------------|
| **定位** | 跨境全链路AI Agent | 亚马逊选品运营一体化 | 阿里生态跨境工具 | 亚马逊数据选品 |
| **核心功能** | 问小盒、AI选品、产品报告、跨境研究 | AI策略选品、三维评分、卖家画像池 | AI选品、AI找商、AI素材、AI询盘 | 产品数据库、关键词优化、Listing生成 |
| **平台覆盖** | TikTok+主流跨境15国 | 亚马逊（美英德为主） | 1688生态对接全平台 | 亚马逊全球站点 |
| **定价模式** | 未公开 | ¥2880/年（个人版） | Token消耗制（低价） | $25-199/月 |
| **差异化** | 实时联网情报、跨境研究大脑 | 亿级卖家商业思维融合 | 1688供应链直连、价格优势 | 11年数据积累、5亿+ASIN |
| **目标用户** | TikTok卖家、新手小白 | 亚马逊中小卖家 | 内贸出海商家、新手 | 亚马逊全类型卖家 |
| **融资背景** | 未公开 | 未公开 | 阿里巴巴内部项目 | 独立公司运营 |

### 2.2 竞品核心能力解析

#### GoBox
- **技术亮点**：基于跨境私有AI模型，秒级联网情报更新
- **产品矩阵**：问小盒（知识问答）→ AI选品 → 产品报告 → 跨境研究
- **竞争优势**：TikTok生态深度绑定，15个主要商业化国家覆盖
- **弱点**：品牌知名度较低，定价策略不透明

#### 卖家穿海
- **技术亮点**：独创三维评分体系（卖家/类目/产品）、卖家群体画像池
- **产品矩阵**：选品 → 查专利 → 找货源 → 运营策略
- **竞争优势**：亿级卖家经验融入，亚马逊平台深度覆盖
- **弱点**：仅限亚马逊生态，定价偏高

#### 1688遨虾
- **技术亮点**：阿里AI能力加持，1688供应链数据直连
- **产品矩阵**：选品Agent → 找商Agent → 询盘Agent → 素材Agent → 咨询Agent
- **竞争优势**：阿里云成本优势明显、供应链资源无限
- **弱点**：生态封闭性、依赖阿里系

#### Jungle Scout
- **技术亮点**：AccuSales算法、11年数据积累
- **产品矩阵**：Cobalt（品牌级） + Catalyst（卖家级）
- **竞争优势**：亚马逊生态最成熟、全球品牌信任度高
- **弱点**：AI能力相对薄弱、价格偏高

### 2.3 突围策略建议

**差异化切入点**：

1. **聚焦TikTok Shop新手卖家**
   - 相比Jungle Scout专注亚马逊、卖家穿海深耕亚马逊，TikTok新手卖家缺乏专属工具
   - GoBox已有布局，但品牌和功能深度不足

2. **AI原生设计**
   - 现有工具多为"AI+传统功能"嫁接
   - 可探索纯对话式交互、一键端到端执行

3. **合规赋能**
   - 东南亚各国监管收紧，合规需求爆发
   - 内置各国合规指南、自动合规检测

4. **轻量化定价**
   - 对标1688遨虾的低价策略
   - 采用订阅+Token消耗混合模式

### 3.2 各国合规要求

#### 泰国
- **产品认证**：TISI认证（电子产品需强制）
- **保健品/化妆品**：需FDA注册编号公示
- **平台监管**：19家电商平台列入特殊监管名单
- **数据合规**：PDPA（类GDPR）

#### 越南
- **主体资质**：需设立代表处或授权代表
- **商品信息**：强制信息透明化
- **数据合规**：2026年1月《个人数据保护法》生效
- **数据本地化**：严格，需本地存储24个月

#### 印尼
- **PT PMA**：外资需注册外资投资公司
- **税务申报**：需配合平台代扣代缴
- **数据合规**：PDP Law（类GDPR）

#### 马来西亚
- **SST申报**：每月15日前完成
- **数据合规**：金融服务/电信数据本地化

#### 菲律宾
- **产品认证**：PS/ICC/FDA/BAI认证（按品类）
- **信任标志**：强制电子商务信任标志
- **数据合规**：DPA

| 进展 | 内容 | 影响 |

| QRIS跨境 | 印尼与中国UPI对接 | 手续费可降至1%以下 |

## 六、关键结论

### 核心发现

1. **市场规模**：TikTok Shop东南亚GMV 456亿美元，年增长100%，市场处于高速扩张期
2. **AI渗透率**：跨境电商AI工具渗透率已达78%，但深度应用仍有空间
3. **新手痛点**：选品迷茫、合规不懂是核心痛点，现有工具针对性不足
4. **政策窗口**：东南亚合规趋严，卖家合规需求爆发

### 战略建议

1. **切入时机**：现在——TikTok Shop高速增长期 + 合规需求爆发期
2. **差异化方向**：聚焦TikTok Shop + 新手卖家 + 合规赋能
3. **快速验证**：6个月内MVP + 种子轮融资 + 首批100用户
4. **长期壁垒**：数据积累 + 供应链整合 + 合规模块
