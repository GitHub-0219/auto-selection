## 一、事件命名规范

### 1.1 命名原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **模块前缀** | 使用产品模块英文缩写 | `reg_`, `prod_`, `pay_`, `course_` |
| **动作类型** | 动宾结构，动词在前 | `click`, `view`, `submit`, `success`, `fail` |
| **命名格式** | 小写字母 + 下划线 | `reg_page_view`, `pay_initiate` |
| **语义清晰** | 名称即含义，无需解释 | `course_chapter_complete` |

### 1.2 命名速查表

| 模块 | 前缀 | 示例 |
|------|------|------|
| 用户注册 | `reg_` | `reg_page_view`, `reg_form_submit` |
| 产品使用 | `prod_` | `prod_category_select`, `prod_analysis_submit` |
| 支付 | `pay_` | `pay_page_view`, `pay_success` |
| 课程学习 | `course_` | `course_list_view`, `course_complete` |
| 通用 | `common_` | `common_share`, `common_feedback` |

### 1.3 事件类型定义

| 类型 | 含义 | 触发时机 |
|------|------|----------|
| `page_view` | 页面浏览 | 进入页面时 |
| `click` | 按钮点击 | 点击可交互元素 |
| `submit` | 表单提交 | 提交表单数据 |
| `success` | 操作成功 | 业务流程成功完成 |
| `fail` | 操作失败 | 业务流程异常终止 |
| `start` | 流程开始 | 启动某项操作 |
| `complete` | 流程完成 | 结束某项操作 |
| `stay` | 停留时长 | 页面/功能使用时长 |

## 二、核心埋点事件设计

### 2.1 用户注册流程

#### 事件列表

| 事件ID | 事件名称 | 触发时机 | 页面/位置 |
|--------|----------|----------|-----------|
| `reg_page_view` | 注册页浏览 | 用户进入注册页面 | 注册引导页 |
| `reg_method_select` | 注册方式选择 | 选择手机/邮箱/第三方登录 | 注册页 |
| `reg_form_fill` | 注册表单填写 | 输入注册信息 | 注册表单 |
| `reg_verify_send` | 验证码发送 | 点击发送验证码 | 注册表单 |
| `reg_verify_success` | 验证码验证成功 | 验证码校验通过 | 注册表单 |
| `reg_verify_fail` | 验证码验证失败 | 验证码错误/过期 | 注册表单 |
| `reg_submit` | 注册表单提交 | 点击注册按钮 | 注册表单 |
| `reg_success` | 注册成功 | 账号创建成功 | 注册完成页 |
| `reg_fail` | 注册失败 | 注册异常终止 | - |

#### 参数说明

```json
{
  "event": "reg_success",
  "params": {
    "register_method": "string",      // 注册方式：phone/email/wechat/apple
    "channel_source": "string",        // 来源渠道：organic/ad/circle
    "country_code": "string",          // 手机国际区号
    "device_type": "string",           // 设备类型：ios/android/h5/web
    "page_source": "string",           // 来源页面
    "time_cost": "number"              // 填写耗时（秒）
  }
}
```

#### 注册漏斗

```
注册页浏览 → 注册方式选择 → 表单填写 → 验证码发送 → 验证码验证 → 注册提交 → 注册成功
    ↓             ↓              ↓            ↓              ↓            ↓
   100%          85%            75%          60%            55%          50%         48%
```

## 三、关键指标定义

### 3.1 用户指标

| 指标名称 | 计算公式 | 数据来源 | 更新频率 |
|----------|----------|----------|----------|
| **DAU** | 日活跃用户数（去重） | `common_app_start` / `common_page_view` | 每日 |
| **MAU** | 月活跃用户数（去重） | `common_app_start` / `common_page_view` | 每日 |
| **DAU/MAU** | DAU / MAU × 100% | 计算得出 | 每日 |
| **新增用户** | 当日新注册用户数 | `reg_success` | 每日 |
| **活跃用户** | 当日有任意事件的用户 | 事件去重 | 每日 |
| **沉默用户** | 30天无活跃用户 | 历史数据计算 | 每日 |
| **流失用户** | 30天未登录且未付费 | 历史数据计算 | 每日 |

### 3.2 转化指标

| 指标名称 | 计算公式 | 基准值 | 目标值 |
|----------|----------|--------|--------|
| **注册转化率** | 注册成功数 / 访问注册页数 | 40% | 55% |
| **表单填写率** | 开始填写 / 访问注册页 | 70% | 80% |
| **验证码发送率** | 发送验证码 / 填写表单 | 85% | 90% |
| **注册完成率** | 注册成功 / 验证码发送 | 90% | 95% |
| **激活率** | 首次使用核心功能 / 注册用户 | 30% | 50% |
| **付费转化率** | 付费用户数 / 访问会员页数 | 3% | 8% |
| **新用户付费率** | 首单用户 / 新增用户 | 5% | 10% |
| **二次付费率** | 复购用户 / 总付费用户 | 20% | 35% |

### 3.3 功能使用指标

| 功能模块 | 指标名称 | 计算方式 | 基准值 | 目标值 |
|----------|----------|----------|--------|--------|
| **选品分析** | 使用率 | 使用人数 / DAU | 25% | 40% |
| | 人均使用次数 | 总使用次数 / 使用人数 | 1.5次 | 2次 |
| | 分析成功率 | 成功数 / 发起数 | 85% | 95% |
| | 平均分析时长 | 总时长 / 成功数 | 30秒 | 15秒 |
| **翻译功能** | 使用率 | 使用人数 / DAU | 15% | 30% |
| | 人均翻译次数 | 总次数 / 使用人数 | 3次 | 5次 |
| **Listing生成** | 使用率 | 使用人数 / DAU | 10% | 25% |
| | 生成成功率 | 成功数 / 发起数 | 90% | 98% |
| | 发布成功率 | 发布数 / 生成数 | 60% | 75% |
| **课程学习** | 学习率 | 学习人数 / DAU | 20% | 35% |
| | 人均学习时长 | 总时长 / 学习人数 | 15分钟 | 30分钟 |
| | 课程完成率 | 完成人数 / 开始人数 | 25% | 45% |

### 3.4 留存指标

| 留存类型 | 计算公式 | 基准值 | 目标值 |
|----------|----------|--------|--------|
| **次日留存** | Day N+1 活跃 / Day 1 新增 × 100% | 35% | 50% |
| **3日留存** | Day 3 活跃 / Day 1 新增 × 100% | 20% | 35% |
| **7日留存** | Day 7 活跃 / Day 1 新增 × 100% | 15% | 25% |
| **14日留存** | Day 14 活跃 / Day 1 新增 × 100% | 10% | 18% |
| **30日留存** | Day 30 活跃 / Day 1 新增 × 100% | 5% | 12% |
| **付费留存** | Day N 付费 / Day 1 新增 × 100% | 3% | 8% |

### 3.5 收入指标

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| **GMV** | 支付成功订单金额之和 | 总成交金额 |
| **ARPU** | GMV / 付费用户数 | 每付费用户平均收入 |
| **ARPPU** | GMV / 付费订单数 | 每付费订单平均金额 |
| **LTV** | 用户生命周期总价值 | 预测用户总贡献 |
| **CAC** | 获客成本 / 获客数量 | 单用户获取成本 |
| **ROI** | LTV / CAC | 投资回报率 |

### 5.1 SDK 初始化

```javascript
// analytics-sdk.js - 数据埋点 SDK 封装

class AnalyticsSDK {
  constructor(config) {
    this.appId = config.appId || 'ai_cross_border_accelerator';
    this.serverUrl = config.serverUrl || 'https://analytics.example.com/collect';
    this.userId = null;
    this.sessionId = this.generateSessionId();
    this.isDebug = config.isDebug || false;
    
    // 页面信息
    this.pageContext = {
      page_name: '',
      page_url: '',
      referrer: ''
    };
  }

  // 初始化
  init() {
    // 获取或生成用户ID
    this.userId = this.getUserId();
    
    // 监听页面切换
    this.bindPageEvents();
    
    // 监听用户行为
    this.bindUserEvents();
    
    // 发送初始化成功事件
    this.track('common_app_start', {
      app_version: this.getAppVersion(),
      platform: this.getPlatform(),
      screen_width: window.innerWidth,
      screen_height: window.innerHeight
    });
    
    this.log('SDK Initialized', { userId: this.userId });
  }

  // 通用事件埋点
  track(eventName, params = {}) {
    const eventData = {
      event: eventName,
      timestamp: Date.now(),
      user_id: this.userId,
      session_id: this.sessionId,
      device_id: this.getDeviceId(),
      platform: this.getPlatform(),
      app_version: this.getAppVersion(),
      ...this.pageContext,
      ...params
    };

    this.send(eventData);
    return eventData;
  }

  // 发送数据
  send(data) {
    if (this.isDebug) {
      console.log('[Analytics]', data);
    }

    // 发送数据到服务端
    navigator.sendBeacon?.(this.serverUrl, JSON.stringify(data))
      || fetch(this.serverUrl, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(data),
        keepalive: true
      }).catch(err => {
        console.error('Analytics send failed:', err);
      });
  }

  // 页面浏览埋点
  trackPageView(pageName, pageParams = {}) {
    return this.track('common_page_view', {
      page_name: pageName,
      page_url: window.location.href,
      referrer: document.referrer,
      stay_duration: 0, // 需配合页面离开事件计算
      ...pageParams
    });
  }

  // 点击事件埋点
  trackClick(elementId, elementName, clickParams = {}) {
    return this.track('common_click', {
      element_id: elementId,
      element_name: elementName,
      element_text: elementId ? document.getElementById(elementId)?.innerText : '',
      ...clickParams
    });
  }

  // 表单提交埋点
  trackFormSubmit(formId, formName, formParams = {}) {
    return this.track('common_form_submit', {
      form_id: formId,
      form_name: formName,
      ...formParams
    });
  }

  // 生成Session ID
  generateSessionId() {
    return `${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
  }

  // 获取用户ID
  getUserId() {
    let userId = localStorage.getItem('analytics_user_id');
    if (!userId) {
      userId = `u_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
      localStorage.setItem('analytics_user_id', userId);
    }
    return userId;
  }

  // 获取设备ID
  getDeviceId() {
    let deviceId = localStorage.getItem('analytics_device_id');
    if (!deviceId) {
      deviceId = this.generateUUID();
      localStorage.setItem('analytics_device_id', deviceId);
    }
    return deviceId;
  }

  // 获取平台信息
  getPlatform() {
    const ua = navigator.userAgent.toLowerCase();
    if (/micromessenger/.test(ua)) return 'wechat';
    if (/iphone|ipad|ipod/.test(ua)) return 'ios';
    if (/android/.test(ua)) return 'android';
    return 'web';
  }

  // 获取App版本
  getAppVersion() {
    return window.APP_VERSION || '1.0.0';
  }

  // UUID生成
  generateUUID() {
    return 'xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'.replace(/[xy]/g, c => {
      const r = Math.random() * 16 | 0;
      return (c === 'x' ? r : (r & 0x3 | 0x8)).toString(16);
    });
  }

  // 绑定页面事件
  bindPageEvents() {
    // 页面加载
    window.addEventListener('load', () => {
      this.pageContext.page_url = window.location.href;
      this.pageContext.referrer = document.referrer;
    });

    // 页面离开 - 计算停留时长
    window.addEventListener('beforeunload', () => {
      this.track('common_page_leave', {
        page_name: this.pageContext.page_name,
        stay_duration: Date.now() - this.pageStartTime
      });
    });

    // 路由变化 (SPA)
    if (window.history.pushState) {
      const originalPushState = window.history.pushState;
      window.history.pushState = (...args) => {
        originalPushState.apply(window.history, args);
        this.onRouteChange();
      };
    }

    this.pageStartTime = Date.now();
  }

  // 路由变化处理
  onRouteChange() {
    this.pageStartTime = Date.now();
    // 可通过解析URL获取页面名称
    const path = window.location.pathname;
    const pageName = this.getPageNameFromPath(path);
    this.trackPageView(pageName);
  }

  // 从路径获取页面名称
  getPageNameFromPath(path) {
    const pathMap = {
      '/': 'home',
      '/product': 'product_home',
      '/product/analysis': 'product_analysis',
      '/product/translate': 'product_translate',
      '/product/listing': 'product_listing',
      '/course': 'course_home',
      '/course/detail': 'course_detail',
      '/course/learning': 'course_learning',
      '/member': 'pay_member',
      '/pay/checkout': 'pay_checkout',
      '/pay/success': 'pay_success',
      '/register': 'reg_page'
    };
    return pathMap[path] || 'unknown';
  }

  // 绑定用户行为事件
  bindUserEvents() {
    // 自动埋点：点击事件
    document.addEventListener('click', (e) => {
      const target = e.target;
      if (target.dataset.analytics) {
        this.track('common_click', {
          element_id: target.id || '',
          element_name: target.dataset.analytics,
          element_text: target.innerText?.trim().substring(0, 100)
        });
      }
    });
  }

  // 日志输出
  log(...args) {
    if (this.isDebug) {
      console.log('[Analytics Debug]', ...args);
    }
  }
}

// 导出单例
window.Analytics = new AnalyticsSDK({

  serverUrl: 'https://analytics.example.com/collect',
  isDebug: process.env.NODE_ENV === 'development'
```

### 5.2 注册流程埋点

```javascript
// register-tracker.js - 注册流程埋点

class RegisterTracker {
  constructor(analytics) {
    this.analytics = analytics;
    this.startTime = null;
    this.registerMethod = null;
  }

  // 注册页浏览
  trackPageView(pageSource = '') {
    this.startTime = Date.now();
    this.analytics.track('reg_page_view', {
      page_source: pageSource,
      channel_source: this.getChannelSource(),
      device_type: this.analytics.getPlatform()
    });
  }

  // 注册方式选择
  trackMethodSelect(method) {
    this.registerMethod = method;
    this.analytics.track('reg_method_select', {
      register_method: method, // phone/email/wechat/apple
      page_source: this.getCurrentPage()
    });
  }

  // 表单填写（防抖）
  trackFormFill(formData) {
    this.analytics.track('reg_form_fill', {
      register_method: this.registerMethod,
      fields_completed: Object.keys(formData).length,
      time_cost: Date.now() - this.startTime
    });
  }

  // 验证码发送
  trackVerifySend() {
    this.analytics.track('reg_verify_send', {
      register_method: this.registerMethod,
      country_code: this.getCountryCode(),
      time_cost: Date.now() - this.startTime
    });
  }

  // 验证码验证
  trackVerifyResult(success, failReason = '') {
    this.analytics.track(success ? 'reg_verify_success' : 'reg_verify_fail', {
      register_method: this.registerMethod,
      fail_reason: failReason,
      time_cost: Date.now() - this.startTime
    });
  }

  // 注册提交
  trackSubmit() {
    this.analytics.track('reg_submit', {
      register_method: this.registerMethod,
      time_cost: Date.now() - this.startTime
    });
  }

  // 注册成功
  trackSuccess(userId, userData = {}) {
    this.analytics.track('reg_success', {
      user_id: userId,
      register_method: this.registerMethod,
      channel_source: this.getChannelSource(),
      country_code: this.getCountryCode(),
      device_type: this.analytics.getPlatform(),
      time_cost: Date.now() - this.startTime,
      ...userData
    });
  }

  // 注册失败
  trackFail(failReason, failStage = '') {
    this.analytics.track('reg_fail', {
      register_method: this.registerMethod,
      fail_reason: failReason,
      fail_stage: failStage,
      time_cost: Date.now() - this.startTime
    });
  }

  // 获取渠道来源
  getChannelSource() {
    const params = new URLSearchParams(window.location.search);
    return params.get('channel') || 'organic';
  }

  // 获取国家区号
  getCountryCode() {
    return localStorage.getItem('user_country_code') || '86';
  }

  // 获取当前页面
  getCurrentPage() {
    return window.location.pathname;
  }
}

// 使用示例
const registerTracker = new RegisterTracker(window.Analytics);

// 监听表单变化

  debounce((e) => {
    registerTracker.trackFormFill({ field: e.target.name });
  }, 1000)
);

// 监听注册提交

  e.preventDefault();
  registerTracker.trackSubmit();
  
  // 执行注册逻辑...
  registerUser(formData).then(user => {
    registerTracker.trackSuccess(user.id, user);
  }).catch(err => {
    registerTracker.trackFail(err.code, 'submit');
  });
});

// 防抖函数
function debounce(func, wait) {
  let timeout;
  return function executedFunction(...args) {
    const later = () => {
      clearTimeout(timeout);
      func(...args);
    };
    clearTimeout(timeout);
    timeout = setTimeout(later, wait);
  };
}
```

```javascript
// course-tracker.js - 课程学习埋点

class CourseTracker {
  constructor(analytics) {
    this.analytics = analytics;
    this.videoStartTime = null;
    this.currentVideo = null;
  }

  // 课程首页浏览
  trackHomeView() {
    this.analytics.trackPageView('course_home');
  }

  // 课程详情浏览
  trackDetailView(courseId, courseData) {
    this.analytics.track('course_detail_view', {
      course_id: courseId,
      course_name: courseData.name,
      course_category: courseData.category,
      total_chapters: courseData.chapterCount,
      is_free: courseData.isFree,
      user_is_vip: courseData.userIsVip
    });
  }

  // 视频播放开始
  trackVideoPlay(chapterId, videoData) {
    this.videoStartTime = Date.now();
    this.currentVideo = {
      chapterId,
      courseId: videoData.courseId,
      videoDuration: videoData.duration
    };
    
    this.analytics.track('course_video_play', {
      course_id: videoData.courseId,
      course_name: videoData.courseName,
      chapter_id: chapterId,
      chapter_name: videoData.chapterName,
      video_duration: videoData.duration,
      playback_speed: videoData.playbackSpeed || '1x',
      is_fullscreen: false
    });
  }

  // 视频暂停
  trackVideoPause(watchDuration) {
    if (!this.currentVideo) return;
    
    this.analytics.track('course_video_pause', {
      course_id: this.currentVideo.courseId,
      chapter_id: this.currentVideo.chapterId,
      watch_duration: watchDuration,
      watch_ratio: watchDuration / this.currentVideo.videoDuration
    });
  }

  // 视频继续播放
  trackVideoResume() {
    this.videoStartTime = Date.now();
  }

  // 视频播放完成
  trackVideoComplete(watchDuration, playbackSpeed) {
    if (!this.currentVideo) return;
    
    const watchRatio = watchDuration / this.currentVideo.videoDuration;
    
    this.analytics.track('course_video_complete', {
      course_id: this.currentVideo.courseId,
      chapter_id: this.currentVideo.chapterId,
      video_duration: this.currentVideo.videoDuration,
      watch_duration: watchDuration,
      watch_ratio: watchRatio,
      playback_speed: playbackSpeed,
      is_fullscreen: document.fullscreenElement !== null
    });

    // 如果观看超过80%，视为章节完成
    if (watchRatio >= 0.8) {
      this.trackChapterComplete();
    }
  }

  // 章节开始
  trackChapterStart(chapterId, chapterData) {
    this.analytics.track('course_chapter_start', {
      course_id: chapterData.courseId,
      course_name: chapterData.courseName,
      chapter_id: chapterId,
      chapter_name: chapterData.chapterName,
      chapter_index: chapterData.index,
      total_chapters: chapterData.totalChapters
    });
  }

  // 章节完成
  trackChapterComplete() {
    if (!this.currentVideo) return;
    
    this.analytics.track('course_chapter_complete', {
      course_id: this.currentVideo.courseId,
      chapter_id: this.currentVideo.chapterId
    });
  }

  // 课程完成
  trackCourseComplete(courseData, completionData) {
    this.analytics.track('course_complete', {
      course_id: courseData.courseId,
      course_name: courseData.courseName,
      course_category: courseData.category,
      total_chapters: courseData.chapterCount,
      completed_chapters: completionData.completedChapters,
      total_duration: courseData.totalDuration,
      actual_duration: completionData.actualDuration,
      completion_time: completionData.totalHours,
      quiz_score: completionData.quizScore || null
    });
  }

  // 学习进度更新
  trackProgressUpdate(courseId, progress) {
    this.analytics.track('course_progress_update', {
      course_id: courseId,
      progress_percent: progress.percent,
      current_chapter: progress.currentChapter,
      total_chapters: progress.totalChapters
    });
  }

  // 视频拖动
  trackVideoSeek(fromPosition, toPosition) {
    if (!this.currentVideo) return;
    
    this.analytics.track('course_video_seek', {
      course_id: this.currentVideo.courseId,
      chapter_id: this.currentVideo.chapterId,
      seek_from: fromPosition,
      seek_to: toPosition,
      seek_direction: toPosition > fromPosition ? 'forward' : 'backward'
    });
  }

  // 添加书签
  trackBookmarkAdd(bookmarkData) {
    this.analytics.track('course_bookmark_add', {
      course_id: bookmarkData.courseId,
      chapter_id: bookmarkData.chapterId,
      bookmark_time: bookmarkData.time
    });
  }

  // 添加笔记
  trackNoteAdd(noteData) {
    this.analytics.track('course_note_add', {
      course_id: noteData.courseId,
      chapter_id: noteData.chapterId,
      note_length: noteData.content.length
    });
  }

  // 课程分享
  trackShare(shareData) {
    this.analytics.track('course_share', {
      course_id: shareData.courseId,
      share_channel: shareData.channel, // wechat/moments/weibo/link
      share_platform: shareData.platform
    });
  }
}

const courseTracker = new CourseTracker(window.Analytics);

// 视频播放器封装
class VideoPlayer {
  constructor(videoElement) {
    this.video = videoElement;
    this.lastPosition = 0;
    
    this.video.addEventListener('play', () => {
      courseTracker.trackVideoPlay(this.dataset.chapterId, {
        courseId: this.dataset.courseId,
        courseName: this.dataset.courseName,
        chapterName: this.dataset.chapterName,
        duration: this.video.duration
      });
    });

    this.video.addEventListener('pause', () => {
      courseTracker.trackVideoPause(this.video.currentTime);
    });

    this.video.addEventListener('ended', () => {
      courseTracker.trackVideoComplete(
        this.video.currentTime,
        this.video.playbackRate
      );
    });

    this.video.addEventListener('seeking', () => {
      courseTracker.trackVideoSeek(this.lastPosition, this.video.currentTime);
      this.lastPosition = this.video.currentTime;
    });
  }
}
```
