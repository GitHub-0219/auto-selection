# Auto选品 UI设计规范文档 v1.0

## 1. 设计理念

### 1.1 品牌定位
- **产品名**：Auto选品（AI跨境新手加速器）
- **目标市场**：TikTok Shop 东南亚
- **目标用户**：跨境电商新手卖家
- **核心价值**：AI驱动的智能选品，降低入门门槛

### 1.2 设计风格
- **风格名称**：专业数据风（Professional Data Style）
- **风格参考**：TradingView、FastMoss
- **视觉特征**：深色背景、数据可视化、专业严谨、高效直观

### 1.3 设计原则
1. **数据优先**：核心数据突出展示，辅助信息层级分明
2. **高效操作**：减少操作步骤，一目了然
3. **专业可信**：传达专业感和可信度
4. **本土适配**：考虑东南亚用户习惯

## 7. 动效规范

### 7.1 过渡时长
| 类型 | 时长 | 用途 |
|-----|------|------|
| instant | 0ms | 即时反馈 |
| fast | 150ms | 微交互 |
| normal | 250ms | 状态切换 |
| slow | 400ms | 页面过渡 |

### 7.2 缓动函数
```css
--ease-out: cubic-bezier(0.16, 1, 0.3, 1);
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 7.3 动效场景
- **Hover**：scale(1.02) + 阴影变化
- **点击**：scale(0.98) + 颜色变化
- **加载**：骨架屏 + 脉冲动画
- **成功**：绿色闪烁 + 缩放弹跳

  /* 圆角 */

### 12.2 命名规范
- **BEM命名法**：`block__element--modifier`
- **组件前缀**：`ui-` (ui-button, ui-card)
- **状态前缀**：`is-` / `has-` (is-active, has-error)
