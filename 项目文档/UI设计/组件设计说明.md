# Auto选品 组件设计说明文档 v1.0

## 概述

本文档定义了Auto选品APP的核心UI组件，包括组件的视觉规范、状态定义和使用指南。所有组件均基于深灰(#1a1a2e) + 蓝绿(#0f9b8e)的主题色系。

## 1. 按钮组件

### 1.1 主按钮 (Primary Button)

```
┌─────────────────────────────────────┐
│ 规范名称：btn-primary               │
├─────────────────────────────────────┤
│ 背景色：渐变 #0f9b8e → #13a89e      │
│ 文字色：#ffffff                     │
│ 字体：14px, font-weight: 600       │
│ 内边距：12px 24px                   │
│ 圆角：8px                           │
│ 高度：44px (移动端最小点击区域)     │
└─────────────────────────────────────┘
```

**状态定义：**
| 状态 | 效果 |
|-----|------|
| Default | 渐变背景，无阴影 |
| Hover | translateY(-2px), 阴影加深 |
| Active | translateY(0), 阴影减弱 |
| Disabled | opacity: 0.5, cursor: not-allowed |
| Loading | 显示loading图标，文字变为"处理中..." |

**使用场景：** 主要操作，如"立即开始"、"确认支付"、"开始分析"

**使用场景：** 次要操作，如"取消"、"返回"、"查看更多"

## 2. 输入框组件

### 2.1 标准输入框

```
┌─────────────────────────────────────┐
│ 🔍 请输入关键词搜索...              │
└─────────────────────────────────────┘

规范名称：input
宽度：100% (或固定宽度)
高度：44px
背景色：rgba(255,255,255,0.05)
边框：1px solid rgba(255,255,255,0.1)
圆角：8px
内边距：12px 16px
文字色：#ffffff
占位符色：#8b8b9e
```

**状态定义：**
| 状态 | 效果 |
|-----|------|
| Default | 浅色边框 |
| Focus | 边框变为#0f9b8e, 外发光 box-shadow |
| Error | 边框变为#e94560, 显示错误提示 |
| Disabled | opacity: 0.5 |

**Focus状态代码：**
```css
input:focus {
  border-color: #0f9b8e;
  box-shadow: 0 0 0 3px rgba(15, 155, 142, 0.2);
  outline: none;
}
```

## 3. 卡片组件

### 3.1 标准卡片

```
┌─────────────────────────────────────┐
│  ┌──────────────────────────────┐   │
│  │ 卡片标题              [icon] │   │
│  ├──────────────────────────────┤   │
│  │                              │   │
│  │ 卡片内容区域                 │   │
│  │                              │   │
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘

规范名称：card
背景色：#16213e
边框：1px solid rgba(255,255,255,0.08)
圆角：12px
内边距：20px
阴影：0 4px 24px rgba(0,0,0,0.3)
```

**状态定义：**
| 状态 | 效果 |
|-----|------|
| Default | 标准样式 |
| Hover | border-color: rgba(15,155,142,0.3), 阴影加深 |
| Active | scale: 0.98 |
| Selected | 边框变为teal, 添加teal光晕 |

---

### 3.2 数据卡片 (用于统计展示)

```
┌─────────────────────────────────────┐
│ 标题                    趋势箭头 ↑ │
│ ────────────────────────────────── │
│ 12,345           单位              │
│ +15.2% vs 上周期                   │
└─────────────────────────────────────┘

特点：
- 大号数字使用等宽字体
- 趋势使用语义色（绿涨红跌）
- 环比数据使用小字号灰色
```

## 7. 弹窗组件

### 7.1 模态框

```
规范名称：modal
背景色：#16213e
边框：1px solid rgba(255,255,255,0.1)
圆角：16px
阴影：0 16px 48px rgba(0,0,0,0.5)
最大宽度：400px (移动端)
内边距：24px

遮罩层：
背景色：rgba(0,0,0,0.7)
点击遮罩可关闭
```

---

### 7.2 底部抽屉

```
规范名称：bottom-sheet
从底部滑入
圆角：顶部16px
最大高度：屏幕90%
拖动条：宽度40px, 高度4px, 居中, 灰色

动画：
- 进入：translateY(100%) → translateY(0)
- 退出：translateY(0) → translateY(100%)
- 时长：300ms
- 缓动：cubic-bezier(0.16, 1, 0.3, 1)
```

---

### 7.3 Toast提示

```
规范名称：toast
位置：顶部居中 (或底部)
背景色：半透明深色
圆角：8px
内边距：12px 20px
阴影：0 4px 12px rgba(0,0,0,0.3)
自动消失：3000ms

类型：
- Success: 左侧绿色图标
- Error: 左侧红色图标
- Warning: 左侧黄色图标
- Info: 左侧蓝色图标
```
