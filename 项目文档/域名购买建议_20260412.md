## 七、购买注意事项

### 7.1 省钱建议

| 建议 | 说明 |
|------|------|
| ✅ 使用新账号 | 新用户可享首年低价优惠 |
| ✅ 一次购买多年 | 锁定当前价格，避免涨价 |
| ✅ 关注促销活动 | 双11、618等节点有额外优惠 |
| ✅ 对比多家平台 | 不同平台价格可能有差异 |

### 7.2 避坑指南

| ⚠️ 注意事项 | 说明 |
|-------------|------|
| 警惕首年低价 | 部分平台首年¥1-9，但续费¥80+ |
| 查看续费价格 | 重点关注续费价格而非首年价 |
| 确认实名要求 | 国内域名必须完成实名认证 |
| 开启自动续费 | 避免域名过期被抢注 |

### 7.3 安全建议

```
1. 开启双因素认证（2FA）
2. 开启域名锁定（Registrar Lock）
3. 启用WHOIS隐私保护
4. 设置到期提醒
```
