## 🚀 快速上手指南

### 第一步：访问产品

打开浏览器，访问：**http://localhost:3000**

### 第二步：登录/注册

**方式一：使用演示账号**
```
邮箱：demo@example.com
密码：demo123
```

**方式二：注册新账号**
1. 点击「免费试用」
2. 输入邮箱和密码
3. 完成注册

### 第三步：体验AI选品

1. 登录后进入「用户中心」
2. 点击「AI智能选品」
3. 输入商品关键词（如：蓝牙耳机、智能手表）
4. 点击「开始分析」
5. 查看AI推荐的商品和市场洞察

### 第四步：探索其他功能

- **多语言翻译**：在翻译功能中输入商品信息，选择目标语言
- **智能定价**：输入成本和目标市场，获取AI定价建议
- **培训课程**：学习跨境电商入门知识

## 🔧 故障排查

### 问题：页面无法访问

**解决方案**：
1. 检查后端服务是否运行：`curl http://localhost:3001/api/health`
2. 检查前端服务是否运行：`curl http://localhost:3000`
3. 重启服务：
   ```bash
   cd ./项目/frontend && npm run dev
   cd ./项目/backend && npm run mock
   ```

### 问题：登录失败

**解决方案**：
1. 使用演示账号：`demo@example.com` / `demo123`
2. 清除浏览器缓存
3. 检查网络连接

### 问题：AI选品无响应

**解决方案**：
1. 检查API服务状态
2. 等待数据加载（演示模式下可能有延迟）
3. 刷新页面重试
