## 附录：效果图规范

### A.1 图片命名规范

```
{页面类型}_{功能模块}_{序号}.{格式}

示例：
- homepage_v2_0.jpg
- profit_calculator_1.jpg
- ios_water_drop_animation_0.gif
```

### A.2 图片尺寸规范

| 场景 | 推荐尺寸 | 说明 |
|------|----------|------|
| 移动端展示 | 1080×1920px | 标准手机屏幕 |
| 平板展示 | 2048×2732px | iPad Pro尺寸 |
| 桌面端展示 | 1920×1080px | 全高清屏幕 |
| 缩略图 | 540×960px | 列表页使用 |

### A.3 图片格式说明

| 格式 | 适用场景 | 优点 |
|------|----------|------|
| JPG | 照片、渐变背景 | 文件小、兼容性好 |
| PNG | 图标、透明背景 | 支持透明、质量无损 |
| WebP | 现代浏览器 | 文件更小、质量更好 |
| GIF | 动图、序列帧 | 支持动画 |

### A.4 pkg格式说明

`pkg://` 是本项目定义的图片引用协议，表示从项目资源包中引用图片资源。

**引用格式：**
```
pkg://{图片相对路径}
```

**示例：**
