## 一键部署命令（复制执行）


## 验证命令（部署后执行）

```bash
# 1. 检查进程
ps aux | grep node

# 2. 检查端口
ss -tlnp | grep -E "3000|3001"

# 3. 测试后端
curl http://localhost:3001/api/health

# 4. 测试前端
curl http://localhost:3000
```

## 如果端口被占用

```bash
# 查看占用端口的进程
lsof -i:3000
lsof -i:3001

# 杀掉占用进程
kill -9 <PID>

# 或者修改端口后重新启动
```

---

## 如果部署失败

```bash
# 查看后端日志
cat ~/auto-selection/project/logs/backend.log

# 查看前端日志
cat ~/auto-selection/project/logs/frontend.log
```

---

## 停止服务

```bash
cd ~/auto-selection/project
kill $(cat logs/*.pid)
```
